20. Valid Parentheses
Easy

題目描述

給定一個只包括 '('、')'、'{'、'}'、'['、']' 的字串 s，判斷字串是否有效。

有效字串需滿足：

  1. 左括號必須用相同類型的右括號閉合
  2. 左括號必須以正確的順序閉合
  3. 每個右括號都有一個對應的相同類型的左括號
範例 1:
輸入: s = "()"
輸出: true
範例 2:
輸入: s = "()[]{}"
輸出: true
範例 3:
輸入: s = "(]"
輸出: false
範例 4:
輸入: s = "([)]"
輸出: false
解釋: 括號順序不正確
範例 5:
輸入: s = "{[]}"
輸出: true

解題思路

方法：堆疊（Stack）

使用堆疊來追蹤左括號，當遇到右括號時檢查堆疊頂部是否有對應的左括號。

核心思想：

堆疊的「後進先出」特性完美符合括號配對的需求。最近的左括號應該最先被配對。

步驟：

  1. 建立一個空堆疊和一個映射表（右括號 → 左括號）
  2. 遍歷字串中的每個字符：
    • 如果是左括號 (、{、[，push 到堆疊
    • 如果是右括號：
      • 檢查堆疊是否為空（沒有對應的左括號）
      • pop 堆疊頂部元素
      • 檢查 pop 出的左括號是否與當前右括號匹配
      • 如果不匹配，返回 false
  3. 遍歷結束後，檢查堆疊是否為空（所有左括號都被配對）

視覺化範例：s = "([{}])"

字符 '('  → push '('     堆疊: ['(']
字符 '['  → push '['     堆疊: ['(', '[']
字符 '{'  → push '{'     堆疊: ['(', '[', '{']
字符 '}'  → pop '{'，匹配 堆疊: ['(', '[']
字符 ']'  → pop '['，匹配 堆疊: ['(']
字符 ')'  → pop '('，匹配 堆疊: []
堆疊為空 → 返回 true

程式碼實現

class Solution:
    def isValid(self, s: str) -> bool:
        """
        使用堆疊檢查括號是否有效
        時間複雜度: O(n)
        空間複雜度: O(n)
        """
        # 建立堆疊
        stack = []
        
        # 建立右括號到左括號的映射
        mapping = {')': '(', '}': '{', ']': '['}
        
        # 遍歷字串
        for char in s:
            # 如果是右括號
            if char in mapping:
                # 取出堆疊頂部元素（如果堆疊為空，使用 '#' 作為佔位符）
                top_element = stack.pop() if stack else '#'
                
                # 檢查是否匹配
                if mapping[char] != top_element:
                    return False
            else:
                # 如果是左括號，push 到堆疊
                stack.append(char)
        
        # 檢查堆疊是否為空（所有括號都已配對）
        return not stack
#include <string>
#include <stack>
#include <unordered_map>

class Solution {
public:
    /**
     * 使用堆疊檢查括號是否有效
     * 時間複雜度: O(n)
     * 空間複雜度: O(n)
     */
    bool isValid(std::string s) {
        // 建立堆疊
        std::stack<char> stack;
        
        // 建立右括號到左括號的映射
        std::unordered_map<char, char> mapping = {
            {')', '('},
            {'}', '{'},
            {']', '['}
        };
        
        // 遍歷字串
        for (char c : s) {
            // 如果是右括號
            if (mapping.count(c)) {
                // 檢查堆疊是否為空
                if (stack.empty()) {
                    return false;
                }
                
                // 取出堆疊頂部元素
                char top = stack.top();
                stack.pop();
                
                // 檢查是否匹配
                if (mapping[c] != top) {
                    return false;
                }
            } else {
                // 如果是左括號，push 到堆疊
                stack.push(c);
            }
        }
        
        // 檢查堆疊是否為空
        return stack.empty();
    }
};
import java.util.Stack;
import java.util.HashMap;
import java.util.Map;

class Solution {
    /**
     * 使用堆疊檢查括號是否有效
     * 時間複雜度: O(n)
     * 空間複雜度: O(n)
     */
    public boolean isValid(String s) {
        // 建立堆疊
        Stack<Character> stack = new Stack<>();
        
        // 建立右括號到左括號的映射
        Map<Character, Character> mapping = new HashMap<>();
        mapping.put(')', '(');
        mapping.put('}', '{');
        mapping.put(']', '[');
        
        // 遍歷字串
        for (char c : s.toCharArray()) {
            // 如果是右括號
            if (mapping.containsKey(c)) {
                // 取出堆疊頂部元素（如果堆疊為空，使用 '#'）
                char top = stack.isEmpty() ? '#' : stack.pop();
                
                // 檢查是否匹配
                if (mapping.get(c) != top) {
                    return false;
                }
            } else {
                // 如果是左括號，push 到堆疊
                stack.push(c);
            }
        }
        
        // 檢查堆疊是否為空
        return stack.isEmpty();
    }
}

複雜度分析

時間複雜度: O(n)

需要遍歷字串中的每個字符一次，其中 n 是字串的長度。

空間複雜度: O(n)

最壞情況下（全是左括號），堆疊需要存儲所有字符。

重點提示

  • 堆疊特性：後進先出（LIFO）完美符合括號配對的需求
  • 映射表：使用雜湊表存儲括號對應關係，使程式碼更清晰
  • 邊界條件：
    • 遇到右括號時堆疊為空 → false
    • 遍歷結束後堆疊不為空 → false
  • 常見錯誤：忘記最後檢查堆疊是否為空
  • 擴展：這個模式可以用於檢查任何需要配對的符號
