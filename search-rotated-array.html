33. Search in Rotated Sorted Array
Medium

題目描述

整數陣列 nums 按升序排列，陣列中的值互不相同。

在傳遞給函數之前，nums 在預先未知的某個下標 k（0 ≤ k < nums.length）上進行了旋轉，使陣列變為 [nums[k], nums[k+1], ..., nums[n-1], nums[0], nums[1], ..., nums[k-1]]（下標從 0 開始計數）。

例如，[0,1,2,4,5,6,7] 在下標 3 處經旋轉後可能變為 [4,5,6,7,0,1,2]。

給你旋轉後的陣列 nums 和一個整數 target，如果 nums 中存在這個目標值 target，則返回它的下標，否則返回 -1。

你必須設計一個時間複雜度為 O(log n) 的演算法解決此問題。

範例 1:
輸入: nums = [4,5,6,7,0,1,2], target = 0
輸出: 4
範例 2:
輸入: nums = [4,5,6,7,0,1,2], target = 3
輸出: -1
範例 3:
輸入: nums = [1], target = 0
輸出: -1

解題思路

方法：修改版二分搜尋

雖然陣列經過旋轉，但仍然可以使用二分搜尋。關鍵是判斷哪一半是有序的，然後決定在哪一半搜尋。

核心思想：

旋轉後的陣列可以分為兩部分，至少有一半是有序的。通過比較 nums[left]、nums[mid]、nums[right]，可以判斷哪一半是有序的。

關鍵觀察：

  • 如果 nums[left] ≤ nums[mid]，說明左半部分是有序的
  • 否則，右半部分是有序的
  • 在有序的那一半中，可以判斷 target 是否在範圍內

步驟：

  1. 初始化 left = 0，right = n - 1
  2. 當 left ≤ right 時：
    • 計算 mid = (left + right) // 2
    • 如果 nums[mid] == target，返回 mid
    • 判斷哪一半是有序的：
      • 如果左半部分有序（nums[left] ≤ nums[mid]）：
        • 如果 target 在左半部分範圍內，right = mid - 1
        • 否則，left = mid + 1
      • 如果右半部分有序：
        • 如果 target 在右半部分範圍內，left = mid + 1
        • 否則，right = mid - 1
  3. 返回 -1

視覺化範例：nums = [4,5,6,7,0,1,2], target = 0

步驟 1: left=0, right=6, mid=3
       [4, 5, 6, 7, 0, 1, 2]
                   ^
       左半部分有序 [4,5,6,7]，target=0 不在此範圍
       搜尋右半部分

步驟 2: left=4, right=6, mid=5
       [4, 5, 6, 7, 0, 1, 2]
                      ^
       右半部分有序 [0,1,2]，target=0 在此範圍
       搜尋左半部分

步驟 3: left=4, right=4, mid=4
       [4, 5, 6, 7, 0, 1, 2]
                   ^
       nums[4]=0 == target，找到！返回 4

程式碼實現

class Solution:
    def search(self, nums: List[int], target: int) -> int:
        """
        修改版二分搜尋
        時間複雜度: O(log n)
        空間複雜度: O(1)
        """
        left, right = 0, len(nums) - 1
        
        while left <= right:
            mid = (left + right) // 2
            
            # 找到目標值
            if nums[mid] == target:
                return mid
            
            # 判斷哪一半是有序的
            # 左半部分有序
            if nums[left] <= nums[mid]:
                # target 在左半部分的範圍內
                if nums[left] <= target < nums[mid]:
                    right = mid - 1
                else:
                    left = mid + 1
            
            # 右半部分有序
            else:
                # target 在右半部分的範圍內
                if nums[mid] < target <= nums[right]:
                    left = mid + 1
                else:
                    right = mid - 1
        
        return -1
#include <vector>

class Solution {
public:
    /**
     * 修改版二分搜尋
     * 時間複雜度: O(log n)
     * 空間複雜度: O(1)
     */
    int search(std::vector<int>& nums, int target) {
        int left = 0, right = nums.size() - 1;
        
        while (left <= right) {
            int mid = left + (right - left) / 2;
            
            // 找到目標值
            if (nums[mid] == target) {
                return mid;
            }
            
            // 左半部分有序
            if (nums[left] <= nums[mid]) {
                // target 在左半部分的範圍內
                if (nums[left] <= target && target < nums[mid]) {
                    right = mid - 1;
                } else {
                    left = mid + 1;
                }
            }
            
            // 右半部分有序
            else {
                // target 在右半部分的範圍內
                if (nums[mid] < target && target <= nums[right]) {
                    left = mid + 1;
                } else {
                    right = mid - 1;
                }
            }
        }
        
        return -1;
    }
};

                return mid;

                    right = mid - 1;

                    left = mid + 1;

            // 右半部分有序
            else {
                // target 在右半部分的範圍內

複雜度分析

時間複雜度: O(log n)

使用二分搜尋，每次將搜尋範圍縮小一半。

空間複雜度: O(1)

只使用了常數額外空間。

重點提示

  • 關鍵判斷：通過比較 nums[left] 和 nums[mid] 判斷哪一半有序
  • 注意等號：nums[left] ≤ nums[mid] 包含等號，處理只有一個元素的情況
  • 範圍判斷：在有序的那一半中，判斷 target 是否在範圍內
  • 常見錯誤：忘記處理邊界情況或範圍判斷錯誤
