704. Binary Search
Easy

題目描述

給定一個 n 個元素有序的（升序）整數陣列 nums 和一個目標值 target，寫一個函數搜尋 nums 中的 target，如果目標值存在返回下標，否則返回 -1。

範例 1:
輸入: nums = [-1,0,3,5,9,12], target = 9
輸出: 4
解釋: 9 出現在 nums 中並且下標為 4
範例 2:
輸入: nums = [-1,0,3,5,9,12], target = 2
輸出: -1
解釋: 2 不存在 nums 中因此返回 -1
限制條件：
  • 1 ≤ nums.length ≤ 104
  • -104 < nums[i], target < 104
  • nums 中的所有整數互不相同
  • nums 按升序排列

解題思路

方法：二分搜尋（Binary Search）

在有序陣列中搜尋目標值，使用二分搜尋可以在 O(log n) 時間內完成。

核心思想：

每次比較中間元素，根據比較結果將搜尋範圍縮小一半，直到找到目標值或搜尋範圍為空。

步驟：

  1. 初始化左指標 left = 0，右指標 right = n - 1
  2. 當 left ≤ right 時，循環執行：
    • 計算中間位置：mid = (left + right) // 2
    • 如果 nums[mid] == target，返回 mid
    • 如果 nums[mid] < target，說明目標在右半部分，left = mid + 1
    • 如果 nums[mid] > target，說明目標在左半部分，right = mid - 1
  3. 如果循環結束仍未找到，返回 -1

視覺化範例：nums = [-1,0,3,5,9,12], target = 9

步驟 1: left=0, right=5, mid=2
       [-1, 0, 3, 5, 9, 12]
                ^
       nums[2]=3 < 9，搜尋右半部分

步驟 2: left=3, right=5, mid=4
       [-1, 0, 3, 5, 9, 12]
                      ^
       nums[4]=9 == 9，找到！返回 4

關鍵技巧：

  • 使用 mid = left + (right - left) // 2 避免整數溢位
  • 注意邊界條件：left ≤ right（包含等號）
  • 更新指標時：left = mid + 1 或 right = mid - 1

程式碼實現

class Solution:
    def search(self, nums: List[int], target: int) -> int:
        """
        二分搜尋
        時間複雜度: O(log n)
        空間複雜度: O(1)
        """
        left, right = 0, len(nums) - 1
        
        while left <= right:
            # 計算中間位置（避免溢位）
            mid = left + (right - left) // 2
            
            # 找到目標值
            if nums[mid] == target:
                return mid
            
            # 目標在右半部分
            elif nums[mid] < target:
                left = mid + 1
            
            # 目標在左半部分
            else:
                right = mid - 1
        
        # 未找到目標值
        return -1
#include <vector>

class Solution {
public:
    /**
     * 二分搜尋
     * 時間複雜度: O(log n)
     * 空間複雜度: O(1)
     */
    int search(std::vector<int>& nums, int target) {
        int left = 0, right = nums.size() - 1;
        
        while (left <= right) {
            // 計算中間位置（避免溢位）
            int mid = left + (right - left) / 2;
            
            // 找到目標值
            if (nums[mid] == target) {
                return mid;
            }
            
            // 目標在右半部分
            else if (nums[mid] < target) {
                left = mid + 1;
            }
            
            // 目標在左半部分
            else {
                right = mid - 1;
            }
        }
        
        // 未找到目標值
        return -1;
    }
};

複雜度分析

時間複雜度: O(log n)

每次比較後搜尋範圍減半，最多需要 log n 次比較。

空間複雜度: O(1)

只使用了常數額外空間（幾個變數）。

重點提示

  • 前提條件：二分搜尋要求陣列必須是有序的
  • 避免溢位：使用 mid = left + (right - left) / 2 而非 (left + right) / 2
  • 邊界條件：while 條件是 left <= right，包含等號
  • 指標更新：更新時要 +1 或 -1，否則可能死循環
  • 變體問題：
    • 找第一個等於目標值的位置
    • 找最後一個等於目標值的位置
    • 找第一個大於等於目標值的位置
