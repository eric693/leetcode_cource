344. Reverse String
Easy

題目描述

編寫一個函數,反轉一個字串陣列。要求原地修改輸入陣列,不使用額外陣列。

輸入: s = ["h","e","l","l","o"]
輸出: ["o","l","l","e","h"]

程式碼實現

Python - 雙指標解法
class Solution:
    def reverseString(self, s: List[str]) -> None:
        left, right = 0, len(s) - 1
        
        while left < right:
            s[left], s[right] = s[right], s[left]
            left += 1
            right -= 1
C++ - 雙指標解法
class Solution {
public:
    void reverseString(vector<char>& s) {
        int left = 0, right = s.size() - 1;
        
        while (left < right) {
            swap(s[left], s[right]);
            left++;
            right--;
        }
    }
};
