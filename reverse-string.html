344. Reverse String

題目描述

編寫一個函數，其作用是將輸入的字串反轉過來。輸入字串以字符陣列 s 的形式給出。

不要給另外的陣列分配額外的空間，你必須原地修改輸入陣列、使用 O(1) 的額外空間解決這一問題。

範例: 
輸入: s = ["h","e","l","l","o"]
輸出: ["o","l","l","e","h"]

解題思路

方法：雙指標

使用兩個指標，一個指向開頭，一個指向結尾，交換元素並向中間移動。

時間複雜度：O(n)

空間複雜度：O(1)

程式碼實現

class Solution:
    def reverseString(self, s: List[str]) -> None:
        left, right = 0, len(s) - 1
        while left < right:
            s[left], s[right] = s[right], s[left]
            left += 1
            right -= 1
class Solution {
    public void reverseString(char[] s) {
        int left = 0, right = s.length - 1;
        while (left < right) {
            char temp = s[left];
            s[left] = s[right];
            s[right] = temp;
            left++;
            right--;
        }
    }
}
