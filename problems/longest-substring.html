3. Longest Substring Without Repeating Characters
Medium

題目描述

給定一個字串 s，請找出其中不含有重複字符的最長子字串的長度。

範例 1:
輸入: s = "abcabcbb"
輸出: 3
解釋: 因為無重複字符的最長子字串是 "abc"，所以其長度為 3
範例 2:
輸入: s = "bbbbb"
輸出: 1
解釋: 因為無重複字符的最長子字串是 "b"，所以其長度為 1
範例 3:
輸入: s = "pwwkew"
輸出: 3
解釋: 因為無重複字符的最長子字串是 "wke"，所以其長度為 3
     請注意，你的答案必須是子字串的長度，"pwke" 是一個子序列，不是子字串

解題思路

方法：滑動視窗（Sliding Window）+ 雜湊表

使用滑動視窗維護一個不含重複字符的子字串，同時使用雜湊表記錄每個字符最後出現的位置。

核心思想：

維護一個視窗 [left, right]，視窗內的子字串不包含重複字符。當遇到重複字符時，移動 left 指標跳過重複字符。

步驟：

  1. 初始化 left 指標為 0，max_length 為 0
  2. 使用雜湊表 char_index 記錄每個字符最後出現的索引
  3. 用 right 指標遍歷字串：
    • 如果當前字符已在視窗中（char_index[s[right]] >= left）：
      • 將 left 移動到重複字符的下一個位置
    • 更新當前字符的索引
    • 更新最大長度：max_length = max(max_length, right - left + 1)
  4. 返回 max_length

視覺化範例：s = "abcabcbb"

步驟 1: "a"bcabcbb    [left=0, right=0]  長度=1
步驟 2: "ab"cabcbb   [left=0, right=1]  長度=2
步驟 3: "abc"abcbb   [left=0, right=2]  長度=3
步驟 4: a"bca"bcbb   [left=1, right=3]  遇到重複'a'，left移至1，長度=3
步驟 5: ab"cab"cbb   [left=2, right=4]  遇到重複'b'，left移至2，長度=3
步驟 6: abc"abc"bb   [left=3, right=5]  遇到重複'c'，left移至3，長度=3
...最長子字串長度為 3

程式碼實現

class Solution:
    def lengthOfLongestSubstring(self, s: str) -> int:
        """
        滑動視窗 + 雜湊表
        時間複雜度: O(n)
        空間複雜度: O(min(n, m))，m 是字符集大小
        """
        # 記錄字符最後出現的索引
        char_index = {}
        left = 0
        max_length = 0
        
        # right 指標遍歷字串
        for right, char in enumerate(s):
            # 如果字符已在當前視窗中
            if char in char_index and char_index[char] >= left:
                # 移動 left 到重複字符的下一個位置
                left = char_index[char] + 1
            
            # 更新字符的索引
            char_index[char] = right
            
            # 更新最大長度
            max_length = max(max_length, right - left + 1)
        
        return max_length
#include <string>
#include <unordered_map>
#include <algorithm>

class Solution {
public:
    /**
     * 滑動視窗 + 雜湊表
     * 時間複雜度: O(n)
     * 空間複雜度: O(min(n, m))
     */
    int lengthOfLongestSubstring(std::string s) {
        // 記錄字符最後出現的索引
        std::unordered_map<char, int> charIndex;
        int left = 0;
        int maxLength = 0;
        
        // right 指標遍歷字串
        for (int right = 0; right < s.length(); right++) {
            char c = s[right];
            
            // 如果字符已在當前視窗中
            if (charIndex.find(c) != charIndex.end() && 
                charIndex[c] >= left) {
                // 移動 left 到重複字符的下一個位置
                left = charIndex[c] + 1;
            }
            
            // 更新字符的索引
            charIndex[c] = right;
            
            // 更新最大長度
            maxLength = std::max(maxLength, right - left + 1);
        }
        
        return maxLength;
    }
};
import java.util.HashMap;
import java.util.Map;

class Solution {
    /**
     * 滑動視窗 + 雜湊表
     * 時間複雜度: O(n)
     * 空間複雜度: O(min(n, m))
     */
    public int lengthOfLongestSubstring(String s) {
        // 記錄字符最後出現的索引
        Map<Character, Integer> charIndex = new HashMap<>();
        int left = 0;
        int maxLength = 0;
        
        // right 指標遍歷字串
        for (int right = 0; right < s.length(); right++) {
            char c = s.charAt(right);
            
            // 如果字符已在當前視窗中
            if (charIndex.containsKey(c) && charIndex.get(c) >= left) {
                // 移動 left 到重複字符的下一個位置
                left = charIndex.get(c) + 1;
            }
            
            // 更新字符的索引
            charIndex.put(c, right);
            
            // 更新最大長度
            maxLength = Math.max(maxLength, right - left + 1);
        }
        
        return maxLength;
    }
}

複雜度分析

時間複雜度: O(n)

每個字符最多被訪問兩次（一次被 right 指標，一次被 left 指標），所以時間複雜度是 O(n)。

空間複雜度: O(min(n, m))

雜湊表需要儲存字符和索引，最多儲存字符集大小 m（如 ASCII 128 或 Unicode）或字串長度 n 個字符。

重點提示

  • 滑動視窗模式：這是解決子字串/子陣列問題的經典模式
  • 關鍵判斷：需要檢查 char_index[char] >= left，確保字符在當前視窗內
  • 視窗更新：遇到重複字符時，left 跳到重複字符的下一個位置
  • 避免錯誤：不要忘記更新字符的索引（每次都要更新）
  • 優化技巧：可以使用陣列代替雜湊表（如果字符集已知且較小）
