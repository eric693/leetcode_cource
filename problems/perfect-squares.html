279. Perfect Squares
Medium

題目描述

給你一個整數 n，返回和為 n 的完全平方數的最少數量。

完全平方數是一個整數，其值等於另一個整數的平方；換句話說，其值等於一個整數自乘的積。例如，1、4、9 和 16 都是完全平方數，而 3 和 11 不是。

範例 1:
輸入: n = 12
輸出: 3
解釋: 12 = 4 + 4 + 4
範例 2:
輸入: n = 13
輸出: 2
解釋: 13 = 4 + 9
限制條件:
1 <= n <= 10^4

解題思路

核心觀察

這是一個典型的完全背包問題或BFS 最短路徑問題：

  • 完全平方數：1, 4, 9, 16, 25, ...
  • 可以重複使用每個完全平方數
  • 目標：找到最少數量的完全平方數，使其和為 n

關鍵：類似 Coin Change 問題，但硬幣是完全平方數

方法一：動態規劃（推薦）⭐⭐⭐

使用 DP 記錄湊成每個數字所需的最少完全平方數個數。

狀態定義：

  • dp[i]：湊成數字 i 所需的最少完全平方數個數
  • 初始狀態：
    • dp[0] = 0（0 不需要任何數）
    • dp[i] = ∞（其他初始化為無窮大）
  • 轉移方程：
    • 對於每個 i，嘗試所有完全平方數 j²（j² ≤ i）
    • dp[i] = min(dp[i], dp[i - j²] + 1)

視覺化範例：n = 12

完全平方數: 1, 4, 9, 16, ...

初始化:
dp[0] = 0
dp[1..12] = ∞

計算 dp[1]:
  使用 1² = 1: dp[1] = dp[0] + 1 = 1
  dp[1] = 1

計算 dp[2]:
  使用 1² = 1: dp[2] = dp[1] + 1 = 2
  dp[2] = 2

計算 dp[3]:
  使用 1² = 1: dp[3] = dp[2] + 1 = 3
  dp[3] = 3

計算 dp[4]:
  使用 1² = 1: dp[4] = dp[3] + 1 = 4
  使用 2² = 4: dp[4] = dp[0] + 1 = 1 ✓
  dp[4] = 1

計算 dp[5]:
  使用 1² = 1: dp[5] = dp[4] + 1 = 2
  使用 2² = 4: dp[5] = dp[1] + 1 = 2
  dp[5] = 2

...繼續計算...

計算 dp[12]:
  使用 1² = 1: dp[12] = dp[11] + 1 = 4
  使用 2² = 4: dp[12] = dp[8] + 1 = 3 ✓
  使用 3² = 9: dp[12] = dp[3] + 1 = 4
  dp[12] = 3

結果: 3 (12 = 4 + 4 + 4)
方法二：BFS 廣度優先搜尋⭐⭐

將問題視為最短路徑問題，從 0 出發，每次可以加一個完全平方數。

思路：

  • 從 n 開始，每次減去一個完全平方數
  • 使用 BFS 找到最短路徑
  • 第一次到達 0 的層數就是答案
方法三：數學定理（拉格朗日四平方和定理）

根據數學定理，任何正整數都可以表示為最多 4 個完全平方數的和。

  • 如果 n 本身是完全平方數 → 答案是 1
  • 如果 n = 4^k × (8m + 7) → 答案是 4
  • 如果可以用兩個完全平方數表示 → 答案是 2
  • 否則答案是 3

程式碼實現

import math
from collections import deque

class Solution:
    def numSquares(self, n: int) -> int:
        """
        方法一：動態規劃（推薦）
        時間複雜度: O(n * sqrt(n))
        空間複雜度: O(n)
        """
        # dp[i] 表示湊成 i 所需的最少完全平方數個數
        dp = [float('inf')] * (n + 1)
        dp[0] = 0
        
        # 計算所有完全平方數
        squares = [i * i for i in range(1, int(math.sqrt(n)) + 1)]
        
        # 填充 DP 表
        for i in range(1, n + 1):
            for square in squares:
                if square > i:
                    break
                dp[i] = min(dp[i], dp[i - square] + 1)
        
        return dp[n]
    
    def numSquares_bfs(self, n: int) -> int:
        """
        方法二：BFS
        時間複雜度: O(n * sqrt(n))
        空間複雜度: O(n)
        """
        # 生成所有完全平方數
        squares = [i * i for i in range(1, int(math.sqrt(n)) + 1)]
        
        # BFS
        queue = deque([(n, 0)])  # (剩餘數字, 層數)
        visited = {n}
        
        while queue:
            remainder, level = queue.popleft()
            
            # 嘗試所有完全平方數
            for square in squares:
                next_remainder = remainder - square
                
                if next_remainder == 0:
                    return level + 1
                
                if next_remainder > 0 and next_remainder not in visited:
                    visited.add(next_remainder)
                    queue.append((next_remainder, level + 1))
        
        return 0
    
    def numSquares_math(self, n: int) -> int:
        """
        方法三：數學定理（拉格朗日四平方和定理）
        時間複雜度: O(sqrt(n))
        空間複雜度: O(1)
        """
        # 檢查是否為完全平方數
        if int(math.sqrt(n)) ** 2 == n:
            return 1
        
        # 檢查是否符合 4^k * (8m + 7) 形式（答案為 4）
        while n % 4 == 0:
            n //= 4
        if n % 8 == 7:
            return 4
        
        # 檢查是否可以用兩個完全平方數表示
        for i in range(1, int(math.sqrt(n)) + 1):
            remainder = n - i * i
            if int(math.sqrt(remainder)) ** 2 == remainder:
                return 2
        
        # 否則答案是 3
        return 3
#include <vector>
#include <cmath>
#include <algorithm>
#include <queue>
#include <unordered_set>
using namespace std;

class Solution {
public:
    /**
     * 方法一：動態規劃（推薦）
     * 時間複雜度: O(n * sqrt(n))
     * 空間複雜度: O(n)
     */
    int numSquares(int n) {
        vector dp(n + 1, INT_MAX);
        dp[0] = 0;
        
        // 預先計算完全平方數
        vector squares;
        for (int i = 1; i * i <= n; i++) {
            squares.push_back(i * i);
        }
        
        // 填充 DP 表
        for (int i = 1; i <= n; i++) {
            for (int square : squares) {
                if (square > i) break;
                dp[i] = min(dp[i], dp[i - square] + 1);
            }
        }
        
        return dp[n];
    }
    
    /**
     * 方法二：BFS
     * 時間複雜度: O(n * sqrt(n))
     * 空間複雜度: O(n)
     */
    int numSquares_bfs(int n) {
        vector squares;
        for (int i = 1; i * i <= n; i++) {
            squares.push_back(i * i);
        }
        
        queue> q;  // (remainder, level)
        q.push({n, 0});
        unordered_set visited;
        visited.insert(n);
        
        while (!q.empty()) {
            auto [remainder, level] = q.front();
            q.pop();
            
            for (int square : squares) {
                int nextRemainder = remainder - square;
                
                if (nextRemainder == 0) {
                    return level + 1;
                }
                
                if (nextRemainder > 0 && visited.find(nextRemainder) == visited.end()) {
                    visited.insert(nextRemainder);
                    q.push({nextRemainder, level + 1});
                }
            }
        }
        
        return 0;
    }
};
import java.util.*;

class Solution {
    /**
     * 方法一：動態規劃（推薦）
     * 時間複雜度: O(n * sqrt(n))
     * 空間複雜度: O(n)
     */
    public int numSquares(int n) {
        int[] dp = new int[n + 1];
        Arrays.fill(dp, Integer.MAX_VALUE);
        dp[0] = 0;
        
        // 預先計算完全平方數
        List squares = new ArrayList<>();
        for (int i = 1; i * i <= n; i++) {
            squares.add(i * i);
        }
        
        // 填充 DP 表
        for (int i = 1; i <= n; i++) {
            for (int square : squares) {
                if (square > i) break;
                dp[i] = Math.min(dp[i], dp[i - square] + 1);
            }
        }
        
        return dp[n];
    }
    
    /**
     * 方法二：BFS
     * 時間複雜度: O(n * sqrt(n))
     * 空間複雜度: O(n)
     */
    public int numSquares_bfs(int n) {
        List squares = new ArrayList<>();
        for (int i = 1; i * i <= n; i++) {
            squares.add(i * i);
        }
        
        Queue queue = new LinkedList<>();
        queue.offer(new int[]{n, 0});  // [remainder, level]
        Set visited = new HashSet<>();
        visited.add(n);
        
        while (!queue.isEmpty()) {
            int[] current = queue.poll();
            int remainder = current[0];
            int level = current[1];
            
            for (int square : squares) {
                int nextRemainder = remainder - square;
                
                if (nextRemainder == 0) {
                    return level + 1;
                }
                
                if (nextRemainder > 0 && !visited.contains(nextRemainder)) {
                    visited.add(nextRemainder);
                    queue.offer(new int[]{nextRemainder, level + 1});
                }
            }
        }
        
        return 0;
    }
}

複雜度分析

方法一：動態規劃
  • 時間複雜度：O(n × √n) - 外層循環 n 次，內層最多 √n 個完全平方數
  • 空間複雜度：O(n) - DP 陣列
方法二：BFS
  • 時間複雜度：O(n × √n)
  • 空間複雜度：O(n) - 隊列和 visited 集合
方法三：數學定理
  • 時間複雜度：O(√n)
  • 空間複雜度：O(1)

重點提示

  • 問題本質：完全背包問題，物品可以重複使用
  • DP 思路：
    • dp[i] = 湊成 i 的最少完全平方數個數
    • 轉移：嘗試所有可能的完全平方數
    • 類似 Coin Change 問題
  • BFS 思路：
    • 從 n 出發，每次減去一個完全平方數
    • 最先到達 0 的路徑就是最短的
    • 需要 visited 避免重複訪問
  • 三種方法比較：
    • DP：穩定可靠，易理解
    • BFS：也很直觀，性能相近
    • 數學：最快但需要數學知識
  • 優化技巧：
    • 預先計算所有完全平方數
    • 完全平方數最多 √n 個
    • 可以提前 break（square > i）
  • 數學定理（拉格朗日四平方和）：
    • 任何正整數都能表示為最多 4 個完全平方數的和
    • 答案只可能是 1, 2, 3, 或 4
    • 可以用來優化判斷
  • 邊界情況：
    • n = 1 → 1 (1² = 1)
    • n = 4 → 1 (2² = 4)
    • n = 12 → 3 (4 + 4 + 4)
    • n 本身是完全平方數 → 1
  • 常見錯誤：
    • DP 初始值設置錯誤
    • 忘記預先計算完全平方數
    • BFS 忘記使用 visited
  • 面試技巧：
    • 先說 DP 解法（最常見）
    • 可以提到 BFS 的思路
    • 展示 DP 表格填充過程
    • 如果知道數學定理可以加分
  • 延伸思考：
    • 如何列出所有可能的組合？
    • 如果要求具體方案而非最少數量？
    • 如果限制只能用某些完全平方數？
