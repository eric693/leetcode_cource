101. Symmetric Tree
Easy

題目描述

給你一個二元樹的根節點 root，檢查它是否軸對稱。

範例 1:
輸入: root = [1,2,2,3,4,4,3]
輸出: true
    1
   / \
  2   2
 / \ / \
3  4 4  3
範例 2:
輸入: root = [1,2,2,null,3,null,3]
輸出: false
    1
   / \
  2   2
   \   \
   3    3
限制條件:
樹中節點數目在範圍 [1, 1000] 內
-100 <= Node.val <= 100
進階:

你可以運用遞迴和迭代兩種方法解決這個問題嗎？

解題思路

核心觀察

對稱樹的特點：

  • 左子樹和右子樹是鏡像對稱的
  • 左子樹的左節點 = 右子樹的右節點
  • 左子樹的右節點 = 右子樹的左節點

關鍵：同時遍歷左右子樹，比較鏡像位置

方法一：遞迴 ⭐⭐⭐

檢查兩棵樹是否互為鏡像。

鏡像條件：

  1. 兩個節點值相同
  2. left.left 和 right.right 是鏡像
  3. left.right 和 right.left 是鏡像

視覺化檢查過程：

       1
      / \
     2   2
    / \ / \
   3  4 4  3

檢查步驟:
1. 根節點的左右子樹
   左: 2, 右: 2 ✓

2. 檢查 left(2) 和 right(2) 的子節點
   left.left(3) vs right.right(3) ✓
   left.right(4) vs right.left(4) ✓

3. 檢查葉子節點
   都是 null ✓

結果: 對稱 ✓

非對稱範例：

       1
      / \
     2   2
      \   \
      3    3

檢查步驟:
1. 根節點的左右子樹
   左: 2, 右: 2 ✓

2. 檢查 left(2) 和 right(2) 的子節點
   left.left(null) vs right.right(3) ✗
   
結果: 不對稱 ✗
方法二：迭代（使用佇列）⭐⭐

使用 BFS 同時遍歷左右子樹。

核心思想：

  • 使用佇列同時處理左右子樹的節點
  • 每次從佇列取出兩個節點比較
  • 按鏡像順序加入子節點

程式碼實現

# Definition for a binary tree node.
class TreeNode:
    def __init__(self, val=0, left=None, right=None):
        self.val = val
        self.left = left
        self.right = right

class Solution:
    def isSymmetric(self, root: TreeNode) -> bool:
        """
        方法一：遞迴
        時間複雜度: O(n)
        空間複雜度: O(h)，h 是樹的高度
        """
        if not root:
            return True
        
        return self.isMirror(root.left, root.right)
    
    def isMirror(self, left: TreeNode, right: TreeNode) -> bool:
        """檢查兩棵樹是否互為鏡像"""
        # 兩個都是空
        if not left and not right:
            return True
        
        # 只有一個是空
        if not left or not right:
            return False
        
        # 值不同
        if left.val != right.val:
            return False
        
        # 遞迴檢查：
        # left 的左子樹 vs right 的右子樹
        # left 的右子樹 vs right 的左子樹
        return (self.isMirror(left.left, right.right) and 
                self.isMirror(left.right, right.left))
    
    def isSymmetric_v2(self, root: TreeNode) -> bool:
        """
        方法二：迭代（使用佇列）
        時間複雜度: O(n)
        空間複雜度: O(n)
        """
        if not root:
            return True
        
        from collections import deque
        queue = deque([(root.left, root.right)])
        
        while queue:
            left, right = queue.popleft()
            
            # 兩個都是空
            if not left and not right:
                continue
            
            # 只有一個是空或值不同
            if not left or not right or left.val != right.val:
                return False
            
            # 按鏡像順序加入
            queue.append((left.left, right.right))
            queue.append((left.right, right.left))
        
        return True
/**
 * Definition for a binary tree node.
 * struct TreeNode {
 *     int val;
 *     TreeNode *left;
 *     TreeNode *right;
 *     TreeNode() : val(0), left(nullptr), right(nullptr) {}
 *     TreeNode(int x) : val(x), left(nullptr), right(nullptr) {}
 *     TreeNode(int x, TreeNode *left, TreeNode *right) : val(x), left(left), right(right) {}
 * };
 */
#include 
using namespace std;

class Solution {
public:
    bool isSymmetric(TreeNode* root) {
        if (!root) return true;
        return isMirror(root->left, root->right);
    }
    
private:
    bool isMirror(TreeNode* left, TreeNode* right) {
        if (!left && !right) return true;
        if (!left || !right) return false;
        if (left->val != right->val) return false;
        
        return isMirror(left->left, right->right) && 
               isMirror(left->right, right->left);
    }
    
public:
    // 迭代版本
    bool isSymmetric_v2(TreeNode* root) {
        if (!root) return true;
        
        queue> q;
        q.push({root->left, root->right});
        
        while (!q.empty()) {
            auto [left, right] = q.front();
            q.pop();
            
            if (!left && !right) continue;
            if (!left || !right || left->val != right->val) 
                return false;
            
            q.push({left->left, right->right});
            q.push({left->right, right->left});
        }
        
        return true;
    }
};
/**
 * Definition for a binary tree node.
 * public class TreeNode {
 *     int val;
 *     TreeNode left;
 *     TreeNode right;
 *     TreeNode() {}
 *     TreeNode(int val) { this.val = val; }
 *     TreeNode(int val, TreeNode left, TreeNode right) {
 *         this.val = val;
 *         this.left = left;
 *         this.right = right;
 *     }
 * }
 */
import java.util.LinkedList;
import java.util.Queue;

class Solution {
    public boolean isSymmetric(TreeNode root) {
        if (root == null) return true;
        return isMirror(root.left, root.right);
    }
    
    private boolean isMirror(TreeNode left, TreeNode right) {
        if (left == null && right == null) return true;
        if (left == null || right == null) return false;
        if (left.val != right.val) return false;
        
        return isMirror(left.left, right.right) && 
               isMirror(left.right, right.left);
    }
    
    // 迭代版本
    public boolean isSymmetric_v2(TreeNode root) {
        if (root == null) return true;
        
        Queue queue = new LinkedList<>();
        queue.offer(root.left);
        queue.offer(root.right);
        
        while (!queue.isEmpty()) {
            TreeNode left = queue.poll();
            TreeNode right = queue.poll();
            
            if (left == null && right == null) continue;
            if (left == null || right == null || left.val != right.val) 
                return false;
            
            queue.offer(left.left);
            queue.offer(right.right);
            queue.offer(left.right);
            queue.offer(right.left);
        }
        
        return true;
    }
}

複雜度分析

時間複雜度: O(n)

需要遍歷所有節點，n 是節點數量

空間複雜度: O(h) / O(n)

遞迴：O(h) 棧空間，h 是樹高；迭代：O(n) 佇列空間

重點提示

  • 對稱定義：左子樹和右子樹互為鏡像
  • 鏡像檢查：left.left vs right.right, left.right vs right.left
  • 邊界情況：空樹、單節點、只有一側子樹
  • 遞迴 vs 迭代：遞迴更簡潔，迭代避免棧溢位
  • 佇列順序：必須成對取出節點
  • 三種情況：都空（繼續）、一空（false）、值不同（false）
  • 常見錯誤：忘記檢查結構對稱（只檢查值）
  • 相關題目：Same Tree (#100)、Invert Binary Tree (#226)
  • 面試技巧：先說遞迴，再提迭代優化
  • 記憶口訣：「鏡像檢查，交叉比較」
