217. Contains Duplicate
Easy

題目描述

給定一個整數陣列 nums，如果任一值在陣列中出現至少兩次，返回 true；如果每個元素都不相同，則返回 false。

範例 1:
輸入: nums = [1,2,3,1]
輸出: true
範例 2:
輸入: nums = [1,2,3,4]
輸出: false
範例 3:
輸入: nums = [1,1,1,3,3,4,3,2,4,2]
輸出: true
限制條件:
1 <= nums.length <= 10^5
-10^9 <= nums[i] <= 10^9

解題思路

方法一：哈希集合（推薦）⭐

使用 Set 記錄已經出現過的元素，遇到重複元素立即返回 true。

核心思想：

遍歷陣列，每次檢查當前元素是否已在 Set 中。如果在，說明有重複；如果不在，加入 Set。

步驟：

  1. 創建空的 Set
  2. 遍歷陣列：
    • 如果當前元素在 Set 中 → 返回 true
    • 否則將當前元素加入 Set
  3. 遍歷結束後返回 false（沒有重複）

視覺化範例：nums = [1,2,3,1]

初始: Set = {}

步驟 1: 檢查 1
       1 不在 Set 中 → 加入
       Set = {1}

步驟 2: 檢查 2
       2 不在 Set 中 → 加入
       Set = {1, 2}

步驟 3: 檢查 3
       3 不在 Set 中 → 加入
       Set = {1, 2, 3}

步驟 4: 檢查 1
       1 在 Set 中 ✓ 找到重複！
       返回 true
方法二：排序後比較

先排序陣列，然後比較相鄰元素是否相同。

優缺點：

  • 優點：空間複雜度 O(1)（不考慮排序的空間）
  • 缺點：時間複雜度 O(n log n)，比哈希集合慢
方法三：暴力法（不推薦）

雙重循環，比較每對元素。

時間複雜度 O(n²)，面試中應避免。

程式碼實現

class Solution:
    def containsDuplicate(self, nums: List[int]) -> bool:
        """
        方法一：哈希集合（推薦）
        時間複雜度: O(n)
        空間複雜度: O(n)
        """
        seen = set()
        
        for num in nums:
            if num in seen:
                return True
            seen.add(num)
        
        return False
    
    def containsDuplicate_pythonic(self, nums: List[int]) -> bool:
        """
        方法一：Python 簡潔寫法
        """
        return len(nums) != len(set(nums))
    
    def containsDuplicate_sort(self, nums: List[int]) -> bool:
        """
        方法二：排序後比較
        時間複雜度: O(n log n)
        空間複雜度: O(1)
        """
        nums.sort()
        
        for i in range(len(nums) - 1):
            if nums[i] == nums[i + 1]:
                return True
        
        return False
#include <vector>
#include <unordered_set>
#include <algorithm>

class Solution {
public:
    /**
     * 方法一：哈希集合（推薦）
     * 時間複雜度: O(n)
     * 空間複雜度: O(n)
     */
    bool containsDuplicate(vector& nums) {
        unordered_set seen;
        
        for (int num : nums) {
            if (seen.count(num) > 0) {
                return true;
            }
            seen.insert(num);
        }
        
        return false;
    }
    
    /**
     * 方法二：排序後比較
     * 時間複雜度: O(n log n)
     * 空間複雜度: O(1)
     */
    bool containsDuplicate_sort(vector& nums) {
        sort(nums.begin(), nums.end());
        
        for (int i = 0; i < nums.size() - 1; i++) {
            if (nums[i] == nums[i + 1]) {
                return true;
            }
        }
        
        return false;
    }
};
import java.util.HashSet;
import java.util.Arrays;

class Solution {
    /**
     * 方法一：哈希集合（推薦）
     * 時間複雜度: O(n)
     * 空間複雜度: O(n)
     */
    public boolean containsDuplicate(int[] nums) {
        HashSet seen = new HashSet<>();
        
        for (int num : nums) {
            if (seen.contains(num)) {
                return true;
            }
            seen.add(num);
        }
        
        return false;
    }
    
    /**
     * 方法二：排序後比較
     * 時間複雜度: O(n log n)
     * 空間複雜度: O(1)
     */
    public boolean containsDuplicate_sort(int[] nums) {
        Arrays.sort(nums);
        
        for (int i = 0; i < nums.length - 1; i++) {
            if (nums[i] == nums[i + 1]) {
                return true;
            }
        }
        
        return false;
    }
}

複雜度分析

方法一：哈希集合
  • 時間複雜度：O(n) - 遍歷陣列一次
  • 空間複雜度：O(n) - Set 最多存儲 n 個元素
方法二：排序
  • 時間複雜度：O(n log n) - 排序的時間
  • 空間複雜度：O(1) - 原地排序

重點提示

  • 最優解：哈希集合 O(n) 時間
  • Python 技巧：len(nums) != len(set(nums)) 一行搞定
  • 空間換時間：使用 O(n) 空間換取 O(n) 時間
  • 提前返回：找到重複立即返回，不需遍歷完整陣列
  • 邊界情況：
    • 只有一個元素 → false
    • 所有元素相同 → true
    • 空陣列 → 題目保證不會出現
  • 面試技巧：
    • 先說哈希集合 O(n) 解法
    • 提到排序解法作為空間優化
    • 說明為什麼不用暴力法
