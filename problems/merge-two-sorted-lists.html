21. Merge Two Sorted Lists
Easy

題目描述

將兩個升序鏈表合併為一個新的升序鏈表並返回。新鏈表是通過拼接給定的兩個鏈表的所有節點組成的。

範例 1:
輸入: list1 = [1,2,4], list2 = [1,3,4]
輸出: [1,1,2,3,4,4]

解釋:
list1: 1 → 2 → 4
list2: 1 → 3 → 4
合併後: 1 → 1 → 2 → 3 → 4 → 4
範例 2:
輸入: list1 = [], list2 = []
輸出: []
範例 3:
輸入: list1 = [], list2 = [0]
輸出: [0]
限制條件:
兩個鏈表的節點數目範圍是 [0, 50]
-100 <= Node.val <= 100
list1 和 list2 均按非遞減順序排列

解題思路

方法一：迭代（推薦）

使用雙指標技巧，比較兩個鏈表的當前節點，每次將較小的節點接到結果鏈表後面。

核心思想：

維護一個虛擬頭節點（dummy node），用一個指標逐步建立合併後的鏈表。

步驟：

  1. 創建虛擬頭節點 dummy 和當前指標 current
  2. 當兩個鏈表都不為空時：
    • 比較 list1 和 list2 的值
    • 將較小的節點接到 current 後面
    • 移動對應鏈表的指標
    • 移動 current 指標
  3. 將剩餘的非空鏈表直接接到結果後面
  4. 返回 dummy.next

視覺化範例：list1=[1,2,4], list2=[1,3,4]

初始狀態:
list1: 1 → 2 → 4
list2: 1 → 3 → 4
dummy: ∅
current: dummy

步驟1: 比較 1 vs 1，選 list1 的 1
dummy: ∅ → 1
list1: 2 → 4

步驟2: 比較 2 vs 1，選 list2 的 1
dummy: ∅ → 1 → 1
list2: 3 → 4

步驟3: 比較 2 vs 3，選 list1 的 2
dummy: ∅ → 1 → 1 → 2
list1: 4

步驟4: 比較 4 vs 3，選 list2 的 3
dummy: ∅ → 1 → 1 → 2 → 3
list2: 4

步驟5: 比較 4 vs 4，選 list1 的 4
dummy: ∅ → 1 → 1 → 2 → 3 → 4
list1: 空

步驟6: list1 為空，將 list2 剩餘部分接上
dummy: ∅ → 1 → 1 → 2 → 3 → 4 → 4

返回 dummy.next
方法二：遞迴

利用遞迴的思想，每次選擇兩個鏈表中較小的頭節點，然後遞迴處理剩餘部分。

步驟：

  1. 基礎情況：如果任一鏈表為空，返回另一個鏈表
  2. 比較兩個鏈表的頭節點：
    • 如果 list1.val < list2.val，list1.next = mergeTwoLists(list1.next, list2)
    • 否則，list2.next = mergeTwoLists(list1, list2.next)
  3. 返回較小的頭節點

程式碼實現

# Definition for singly-linked list.
# class ListNode:
#     def __init__(self, val=0, next=None):
#         self.val = val
#         self.next = next

class Solution:
    def mergeTwoLists(self, list1: Optional[ListNode], list2: Optional[ListNode]) -> Optional[ListNode]:
        """
        方法一：迭代（推薦）
        時間複雜度: O(m + n)
        空間複雜度: O(1)
        """
        # 創建虛擬頭節點
        dummy = ListNode(0)
        current = dummy
        
        # 當兩個鏈表都不為空時
        while list1 and list2:
            if list1.val < list2.val:
                current.next = list1
                list1 = list1.next
            else:
                current.next = list2
                list2 = list2.next
            current = current.next
        
        # 將剩餘的非空鏈表接上
        current.next = list1 if list1 else list2
        
        return dummy.next
    
    def mergeTwoLists_recursive(self, list1: Optional[ListNode], list2: Optional[ListNode]) -> Optional[ListNode]:
        """
        方法二：遞迴
        時間複雜度: O(m + n)
        空間複雜度: O(m + n) - 遞迴堆疊
        """
        # 基礎情況
        if not list1:
            return list2
        if not list2:
            return list1
        
        # 遞迴合併
        if list1.val < list2.val:
            list1.next = self.mergeTwoLists_recursive(list1.next, list2)
            return list1
        else:
            list2.next = self.mergeTwoLists_recursive(list1, list2.next)
            return list2
/**
 * Definition for singly-linked list.
 * struct ListNode {
 *     int val;
 *     ListNode *next;
 *     ListNode() : val(0), next(nullptr) {}
 *     ListNode(int x) : val(x), next(nullptr) {}
 *     ListNode(int x, ListNode *next) : val(x), next(next) {}
 * };
 */

class Solution {
public:
    /**
     * 方法一：迭代（推薦）
     * 時間複雜度: O(m + n)
     * 空間複雜度: O(1)
     */
    ListNode* mergeTwoLists(ListNode* list1, ListNode* list2) {
        // 創建虛擬頭節點
        ListNode* dummy = new ListNode(0);
        ListNode* current = dummy;
        
        // 當兩個鏈表都不為空時
        while (list1 != nullptr && list2 != nullptr) {
            if (list1->val < list2->val) {
                current->next = list1;
                list1 = list1->next;
            } else {
                current->next = list2;
                list2 = list2->next;
            }
            current = current->next;
        }
        
        // 將剩餘的非空鏈表接上
        current->next = (list1 != nullptr) ? list1 : list2;
        
        ListNode* result = dummy->next;
        delete dummy;  // 釋放虛擬節點記憶體
        return result;
    }
    
    /**
     * 方法二：遞迴
     * 時間複雜度: O(m + n)
     * 空間複雜度: O(m + n)
     */
    ListNode* mergeTwoLists_recursive(ListNode* list1, ListNode* list2) {
        // 基礎情況
        if (list1 == nullptr) return list2;
        if (list2 == nullptr) return list1;
        
        // 遞迴合併
        if (list1->val < list2->val) {
            list1->next = mergeTwoLists_recursive(list1->next, list2);
            return list1;
        } else {
            list2->next = mergeTwoLists_recursive(list1, list2->next);
            return list2;
        }
    }
};
/**
 * Definition for singly-linked list.
 * public class ListNode {
 *     int val;
 *     ListNode next;
 *     ListNode() {}
 *     ListNode(int val) { this.val = val; }
 *     ListNode(int val, ListNode next) { this.val = val; this.next = next; }
 * }
 */

class Solution {
    /**
     * 方法一：迭代（推薦）
     * 時間複雜度: O(m + n)
     * 空間複雜度: O(1)
     */
    public ListNode mergeTwoLists(ListNode list1, ListNode list2) {
        // 創建虛擬頭節點
        ListNode dummy = new ListNode(0);
        ListNode current = dummy;
        
        // 當兩個鏈表都不為空時
        while (list1 != null && list2 != null) {
            if (list1.val < list2.val) {
                current.next = list1;
                list1 = list1.next;
            } else {
                current.next = list2;
                list2 = list2.next;
            }
            current = current.next;
        }
        
        // 將剩餘的非空鏈表接上
        current.next = (list1 != null) ? list1 : list2;
        
        return dummy.next;
    }
    
    /**
     * 方法二：遞迴
     * 時間複雜度: O(m + n)
     * 空間複雜度: O(m + n)
     */
    public ListNode mergeTwoLists_recursive(ListNode list1, ListNode list2) {
        // 基礎情況
        if (list1 == null) return list2;
        if (list2 == null) return list1;
        
        // 遞迴合併
        if (list1.val < list2.val) {
            list1.next = mergeTwoLists_recursive(list1.next, list2);
            return list1;
        } else {
            list2.next = mergeTwoLists_recursive(list1, list2.next);
            return list2;
        }
    }
}

複雜度分析

時間複雜度: O(m + n)

其中 m 和 n 分別是兩個鏈表的長度。每個節點都會被訪問一次。

空間複雜度:
  • 迭代方法：O(1) - 只使用常數個額外變數
  • 遞迴方法：O(m + n) - 遞迴堆疊深度等於鏈表長度之和

重點提示

  • 虛擬頭節點：使用 dummy node 可以簡化邊界條件處理，避免對頭節點的特殊處理
  • 迭代 vs 遞迴：迭代方法空間效率更高，遞迴方法程式碼更簡潔
  • 原地修改：不需要創建新節點，直接修改原鏈表的指標即可
  • 邊界情況：
    • 一個或兩個鏈表為空
    • 兩個鏈表長度差異很大
    • 所有值相等
  • 不要忘記：處理完後要接上剩餘的非空鏈表
  • 面試技巧：
    • 先說清楚思路，再寫代碼
    • 討論邊界條件
    • 可以先寫迭代解法，再優化為遞迴（或反之）
