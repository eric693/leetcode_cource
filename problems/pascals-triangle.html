118. Pascal's Triangle
Easy

題目描述

給定一個非負整數 numRows，生成「楊輝三角」的前 numRows 行。

在楊輝三角中，每個數是它左上方和右上方的數的和。

視覺化:
        1
       1 1
      1 2 1
     1 3 3 1
    1 4 6 4 1
範例 1:
輸入: numRows = 5
輸出: [[1],[1,1],[1,2,1],[1,3,3,1],[1,4,6,4,1]]
範例 2:
輸入: numRows = 1
輸出: [[1]]
限制條件:
1 <= numRows <= 30

解題思路

核心觀察

楊輝三角（Pascal's Triangle）的規律：

  • 第一行只有一個 1
  • 每行的首尾都是 1
  • 中間的數 = 上一行左上 + 上一行右上
  • 第 n 行有 n 個數字

公式：triangle[i][j] = triangle[i-1][j-1] + triangle[i-1][j]

方法：動態規劃 ⭐⭐⭐

逐行構建楊輝三角。

算法步驟：

  1. 創建結果列表
  2. 對每一行 i（從 0 到 numRows-1）：
    • 創建新行，初始化為 [1] * (i+1)
    • 填充中間元素（如果有）
    • row[j] = prevRow[j-1] + prevRow[j]
    • 將新行加入結果

視覺化構建過程：numRows = 5

第 0 行（i=0）:
[1]
→ 只有一個元素，直接是 1

第 1 行（i=1）:
[1, 1]
→ 首尾都是 1，沒有中間元素

第 2 行（i=2）:
初始: [1, ?, 1]
j=1: row[1] = prevRow[0] + prevRow[1] = 1 + 1 = 2
結果: [1, 2, 1]

第 3 行（i=3）:
初始: [1, ?, ?, 1]
j=1: row[1] = prevRow[0] + prevRow[1] = 1 + 2 = 3
j=2: row[2] = prevRow[1] + prevRow[2] = 2 + 1 = 3
結果: [1, 3, 3, 1]

第 4 行（i=4）:
初始: [1, ?, ?, ?, 1]
j=1: row[1] = prevRow[0] + prevRow[1] = 1 + 3 = 4
j=2: row[2] = prevRow[1] + prevRow[2] = 3 + 3 = 6
j=3: row[3] = prevRow[2] + prevRow[3] = 3 + 1 = 4
結果: [1, 4, 6, 4, 1]

最終結果:
[
  [1],
  [1,1],
  [1,2,1],
  [1,3,3,1],
  [1,4,6,4,1]
]

數字的來源：

      1
     / \
    1   1
   / \ / \
  1   2   1
 / \ / \ / \
1   3   3   1

每個數字 = 左上方數字 + 右上方數字

例如：
- 第 2 行的 2 = 上一行的 1 + 1
- 第 3 行的 3 = 上一行的 1 + 2
- 第 4 行的 6 = 上一行的 3 + 3

數學性質：

  • 第 n 行第 k 個數 = C(n, k) = n! / (k! × (n-k)!)
  • 這是組合數公式
  • 例如：第 4 行第 2 個數 = C(4,2) = 6
  • 但直接用上一行計算更簡單

程式碼實現

class Solution:
    def generate(self, numRows: int) -> List[List[int]]:
        """
        動態規劃
        時間複雜度: O(numRows²)
        空間複雜度: O(1)，不計算輸出空間
        """
        result = []
        
        for i in range(numRows):
            # 創建新行，初始化為全 1
            row = [1] * (i + 1)
            
            # 填充中間元素
            for j in range(1, i):
                row[j] = result[i-1][j-1] + result[i-1][j]
            
            result.append(row)
        
        return result
    
    def generate_v2(self, numRows: int) -> List[List[int]]:
        """
        另一種寫法：不預先初始化
        """
        result = []
        
        for i in range(numRows):
            row = []
            for j in range(i + 1):
                if j == 0 or j == i:
                    # 首尾是 1
                    row.append(1)
                else:
                    # 中間元素
                    row.append(result[i-1][j-1] + result[i-1][j])
            result.append(row)
        
        return result

class Solution {

     * 動態規劃
     * 時間複雜度: O(numRows²)
     * 空間複雜度: O(1)
     */
    vector> generate(int numRows) {
        vector> result;
        
        for (int i = 0; i < numRows; i++) {
            vector row(i + 1, 1);
            
            for (int j = 1; j < i; j++) {
                row[j] = result[i-1][j-1] + result[i-1][j];
            }
            
            result.push_back(row);
        }

    }

import java.util.ArrayList;
import java.util.List;

class Solution {
    /**
     * 動態規劃
     * 時間複雜度: O(numRows²)
     * 空間複雜度: O(1)
     */
    public List> generate(int numRows) {
        List> result = new ArrayList<>();
        
        for (int i = 0; i < numRows; i++) {
            List row = new ArrayList<>();
            
            for (int j = 0; j <= i; j++) {
                if (j == 0 || j == i) {
                    row.add(1);
                } else {
                    int sum = result.get(i-1).get(j-1) + result.get(i-1).get(j);
                    row.add(sum);
                }
            }
            
            result.add(row);
        }
        
        return result;
    }
}

複雜度分析

時間複雜度: O(numRows²)

總共需要計算 1 + 2 + 3 + ... + numRows = numRows×(numRows+1)/2 個元素。

空間複雜度: O(1)

不計算輸出空間，只使用常數額外空間。

重點提示

  • 楊輝三角的規律：
    • 對稱性：左右對稱
    • 每行首尾都是 1
    • 每個數 = 左上 + 右上
    • 第 n 行有 n 個數
  • 構建策略：
    • 從第一行開始逐行構建
    • 每行都依賴上一行
    • 這是典型的動態規劃
  • 兩種實現方式：
    • 方式 1：預先初始化為全 1，再填充中間
    • 方式 2：逐個判斷是首尾還是中間
    • 兩種方式效率相當
  • 索引關係：
    • 當前行索引：i
    • 當前位置：j
    • 左上：result[i-1][j-1]
    • 右上：result[i-1][j]
  • 數學背景：
    • 楊輝三角的每個數是組合數
    • 第 n 行第 k 個 = C(n,k)
    • C(n,k) = n! / (k! × (n-k)!)
    • 遞推關係：C(n,k) = C(n-1,k-1) + C(n-1,k)
  • 實際應用：
    • 二項式展開
    • 組合計數
    • 概率論
    • 數值分析
  • 邊界情況：
    • numRows = 1 → [[1]]
    • numRows = 2 → [[1],[1,1]]
    • 首行特殊處理
  • 常見錯誤：
    • 索引越界（訪問 result[i-1]）
    • 忘記首尾是 1
    • 循環範圍錯誤
  • 優化技巧：
    • 利用對稱性，只計算一半
    • 但實際提升不明顯
    • 代碼會變複雜
  • 二項式定理：
    • (a+b)ⁿ 展開式的係數
    • 例如：(a+b)³ = a³ + 3a²b + 3ab² + b³
    • 係數 [1,3,3,1] 正是第 3 行
  • 面試技巧：
    • 先畫出楊輝三角
    • 找出規律
    • 說明遞推關係
    • 寫出清晰的代碼
  • 變形題：
    • Pascal's Triangle II (#119) - 返回第 n 行
    • 需要 O(n) 空間優化
  • 歷史背景：
    • 中國稱為「楊輝三角」
    • 西方稱為「Pascal's Triangle」
    • 實際上更早出現在中國和波斯
  • 記憶口訣：「首尾全是一，中間上兩數之和」
