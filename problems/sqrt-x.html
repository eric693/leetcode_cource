69. Sqrt(x)
Easy

題目描述

給你一個非負整數 x，計算並返回 x 的算術平方根。

由於返回類型是整數，結果只保留整數部分，小數部分將被捨去。

注意：不允許使用任何內建指數函數和運算符，例如 pow(x, 0.5) 或者 x ** 0.5。

範例 1:
輸入: x = 4
輸出: 2
解釋: 4 的平方根是 2，所以返回 2。
範例 2:
輸入: x = 8
輸出: 2
解釋: 8 的平方根是 2.82842...，由於返回類型是整數，小數部分將被捨去。
限制條件:
0 <= x <= 2^31 - 1

解題思路

核心觀察

這道題的關鍵：

  • 找到最大的整數 k，使得 k² ≤ x
  • 答案範圍在 [0, x]
  • 單調遞增：k 越大，k² 越大
  • 適合用二分搜尋

關鍵：這是一個「找最後一個滿足條件的數」的問題

方法一：二分搜尋（推薦）⭐⭐⭐

在 [0, x] 範圍內二分搜尋答案。

核心思想：

  • 答案範圍：[0, x]
  • 對於每個 mid，檢查 mid² 與 x 的關係
  • 如果 mid² ≤ x，答案可能是 mid 或更大
  • 如果 mid² > x，答案一定更小

注意溢位：

  • mid * mid 可能溢位
  • 改用 mid ≤ x / mid 來比較
  • 或使用 long 型別

視覺化範例：x = 8

尋找最大的 k 使得 k² ≤ 8

初始: left = 0, right = 8

迭代 1:
mid = (0 + 8) / 2 = 4
4² = 16 > 8
right = mid - 1 = 3

迭代 2:
left = 0, right = 3
mid = (0 + 3) / 2 = 1
1² = 1 ≤ 8 ✓
left = mid + 1 = 2

迭代 3:
left = 2, right = 3
mid = (2 + 3) / 2 = 2
2² = 4 ≤ 8 ✓
left = mid + 1 = 3

迭代 4:
left = 3, right = 3
mid = (3 + 3) / 2 = 3
3² = 9 > 8
right = mid - 1 = 2

left > right，循環結束
答案 = right = 2 ✓

驗證: 2² = 4 ≤ 8，3² = 9 > 8

為什麼返回 right 而不是 left？

本題要找「最後一個 k² ≤ x 的 k」

循環結束時：
- left: 第一個 k² > x 的位置
- right: 最後一個 k² ≤ x 的位置

所以答案是 right

對比 #35（插入位置）：
- #35 要找第一個 >= target，返回 left
- #69 要找最後一個滿足條件，返回 right
方法二：牛頓迭代法 ⭐⭐

使用牛頓法求方程 f(x) = x² - a = 0 的根。

公式：

x_{n+1} = (x_n + a/x_n) / 2

幾何意義：

  • 用切線逼近曲線
  • 收斂速度很快（平方收斂）
  • 每次迭代精度翻倍

牛頓法範例：x = 8

初始猜測: x₀ = 8

迭代 1:
x₁ = (8 + 8/8) / 2 = (8 + 1) / 2 = 4.5

迭代 2:
x₂ = (4.5 + 8/4.5) / 2 = (4.5 + 1.778) / 2 = 3.139

迭代 3:
x₃ = (3.139 + 8/3.139) / 2 = (3.139 + 2.548) / 2 = 2.843

迭代 4:
x₄ = (2.843 + 8/2.843) / 2 = 2.828

收斂到 √8 ≈ 2.828
取整數部分 = 2 ✓
方法三：暴力法

從 0 開始逐一嘗試。

  • 時間複雜度：O(√x)
  • 會超時，不推薦

程式碼實現

class Solution:
    def mySqrt(self, x: int) -> int:
        """
        方法一：二分搜尋
        時間複雜度: O(log x)
        空間複雜度: O(1)
        """
        if x == 0:
            return 0
        
        left, right = 1, x
        
        while left <= right:
            mid = left + (right - left) // 2
            
            # 避免溢位：mid * mid > x 改為 mid > x / mid
            if mid <= x // mid:
                left = mid + 1
            else:
                right = mid - 1
        
        return right
    
    def mySqrt_v2(self, x: int) -> int:
        """
        方法一：二分搜尋（使用 long）
        """
        if x == 0:
            return 0
        
        left, right = 1, x
        
        while left <= right:
            mid = left + (right - left) // 2
            square = mid * mid
            
            if square == x:
                return mid
            elif square < x:
                left = mid + 1
            else:
                right = mid - 1
        
        return right
    
    def mySqrt_newton(self, x: int) -> int:
        """
        方法二：牛頓迭代法
        時間複雜度: O(log x)
        空間複雜度: O(1)
        """
        if x == 0:
            return 0
        
        # 初始猜測
        r = x
        
        while r > x // r:
            # 牛頓公式
            r = (r + x // r) // 2
        
        return r
class Solution {
public:
    /**
     * 方法一：二分搜尋
     * 時間複雜度: O(log x)
     * 空間複雜度: O(1)
     */
    int mySqrt(int x) {
        if (x == 0) return 0;
        
        long left = 1, right = x;
        
        while (left <= right) {
            long mid = left + (right - left) / 2;
            long square = mid * mid;
            
            if (square == x) {
                return mid;
            } else if (square < x) {
                left = mid + 1;
            } else {
                right = mid - 1;
            }
        }
        
        return right;
    }
    
    /**
     * 方法二：牛頓迭代法
     * 時間複雜度: O(log x)
     * 空間複雜度: O(1)
     */
    int mySqrt_newton(int x) {
        if (x == 0) return 0;
        
        long r = x;
        
        while (r > x / r) {
            r = (r + x / r) / 2;
        }
        
        return r;
    }
};
class Solution {
    /**
     * 方法一：二分搜尋
     * 時間複雜度: O(log x)
     * 空間複雜度: O(1)
     */
    public int mySqrt(int x) {
        if (x == 0) return 0;
        
        long left = 1, right = x;
        
        while (left <= right) {
            long mid = left + (right - left) / 2;
            long square = mid * mid;
            
            if (square == x) {
                return (int) mid;
            } else if (square < x) {
                left = mid + 1;
            } else {
                right = mid - 1;
            }
        }
        
        return (int) right;
    }
    
    /**
     * 方法二：牛頓迭代法
     * 時間複雜度: O(log x)
     * 空間複雜度: O(1)
     */
    public int mySqrt_newton(int x) {
        if (x == 0) return 0;
        
        long r = x;
        
        while (r > x / r) {
            r = (r + x / r) / 2;
        }
        
        return (int) r;
    }
}

複雜度分析

時間複雜度: O(log x)

二分搜尋和牛頓法都是對數時間。

空間複雜度: O(1)

只使用常數額外空間。

重點提示

  • 二分搜尋要點：
    • 搜尋範圍：[1, x]（0 要特殊處理）
    • 返回 right（最後一個滿足條件的）
    • 注意溢位問題
  • 溢位處理：
    • 方法 1：mid * mid 可能溢位 → 用 long
    • 方法 2：改用 mid ≤ x / mid
    • 方法 3：mid ≤ x // mid（Python 整除）
  • 為什麼返回 right？
    • 要找最後一個 k² ≤ x
    • 循環結束：left 超過了答案
    • right 停在正確答案上
  • 牛頓法優勢：
    • 收斂速度更快（平方收斂）
    • 迭代次數更少
    • 適合高精度計算
  • 牛頓法原理：
    • 求解 f(x) = x² - a = 0
    • f'(x) = 2x
    • x_{n+1} = x_n - f(x_n)/f'(x_n)
    • = x_n - (x_n² - a)/(2x_n)
    • = (x_n + a/x_n) / 2
  • 邊界情況：
    • x = 0 → 0
    • x = 1 → 1
    • x = 2 → 1
    • 完全平方數（4, 9, 16...）
  • 常見錯誤：
    • 返回 left 而不是 right
    • 忘記處理溢位
    • 初始 right = x/2（錯誤，應該是 x）
    • 牛頓法除以 0
  • 優化技巧：
    • right 可以優化為 x/2 + 1（√x ≤ x/2 對 x ≥ 4）
    • 但要注意小數字的情況
    • 實際提升不明顯
  • 二分搜尋變形：
    • #35：找插入位置（第一個 >=）
    • #69：找平方根（最後一個 ≤）
    • 同樣是二分，返回值不同
  • 面試技巧：
    • 先提二分搜尋（最常見）
    • 說明溢位處理
    • 可以提牛頓法（加分項）
    • 解釋為什麼返回 right
