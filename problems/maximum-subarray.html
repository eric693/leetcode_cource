53. Maximum Subarray
Medium

題目描述

給定一個整數陣列 nums，找出一個具有最大和的連續子陣列（子陣列最少包含一個元素），返回其最大和。

子陣列是陣列中的一個連續部分。

範例 1:
輸入: nums = [-2,1,-3,4,-1,2,1,-5,4]
輸出: 6
解釋: 連續子陣列 [4,-1,2,1] 的和最大，為 6
範例 2:
輸入: nums = [1]
輸出: 1
解釋: 只有一個元素，最大和就是 1
範例 3:
輸入: nums = [5,4,-1,7,8]
輸出: 23
解釋: 整個陣列的和最大，為 23
限制條件:
1 <= nums.length <= 10^5
-10^4 <= nums[i] <= 10^4
進階:

如果你已經實現 O(n) 的解法，嘗試使用分治法解決，時間複雜度為 O(n log n)。

解題思路

方法一：Kadane's Algorithm（推薦）⭐

這是解決最大子陣列和問題的經典演算法，時間複雜度 O(n)，空間複雜度 O(1)。

核心思想：

對於每個位置，決定是「繼續累加」還是「重新開始」：

  • 如果當前累加和 > 0，繼續累加當前元素
  • 如果當前累加和 ≤ 0，從當前元素重新開始

步驟：

  1. 初始化 current_sum = 0, max_sum = -∞
  2. 遍歷陣列每個元素：
    • current_sum = max(num, current_sum + num)
    • max_sum = max(max_sum, current_sum)
  3. 返回 max_sum

視覺化範例：nums = [-2,1,-3,4,-1,2,1,-5,4]

i=0: num=-2, current=-2, max=-2
i=1: num=1,  current=1,  max=1   (重新開始,因為-2+1<1)
i=2: num=-3, current=-2, max=1
i=3: num=4,  current=4,  max=4   (重新開始,因為-2+4<4)
i=4: num=-1, current=3,  max=4   (繼續累加,4+(-1)=3>-1)
i=5: num=2,  current=5,  max=5
i=6: num=1,  current=6,  max=6   ⭐ 最大值
i=7: num=-5, current=1,  max=6
i=8: num=4,  current=5,  max=6

答案: 6 (子陣列 [4,-1,2,1])
方法二：動態規劃

定義 dp[i] 為以 nums[i] 結尾的最大子陣列和。

狀態轉移方程：

dp[i] = max(nums[i], dp[i-1] + nums[i])

解釋：

對於每個位置 i，要麼自己單獨成為一個子陣列，要麼接在前面的最大子陣列後面。

方法三：分治法（進階）

將陣列分成左右兩部分，最大子陣列和可能在：

  • 完全在左半部分
  • 完全在右半部分
  • 跨越中間點

時間複雜度：O(n log n)

程式碼實現

class Solution:
    def maxSubArray(self, nums: List[int]) -> int:
        """
        方法一：Kadane's Algorithm（推薦）
        時間複雜度: O(n)
        空間複雜度: O(1)
        """
        current_sum = 0
        max_sum = float('-inf')
        
        for num in nums:
            # 決定是繼續累加還是重新開始
            current_sum = max(num, current_sum + num)
            # 更新最大值
            max_sum = max(max_sum, current_sum)
        
        return max_sum
    
    def maxSubArray_dp(self, nums: List[int]) -> int:
        """
        方法二：動態規劃
        時間複雜度: O(n)
        空間複雜度: O(n)
        """
        n = len(nums)
        # dp[i] 表示以 nums[i] 結尾的最大子陣列和
        dp = [0] * n
        dp[0] = nums[0]
        
        for i in range(1, n):
            dp[i] = max(nums[i], dp[i-1] + nums[i])
        
        return max(dp)
    
    def maxSubArray_dp_optimized(self, nums: List[int]) -> int:
        """
        方法二：動態規劃（空間優化）
        時間複雜度: O(n)
        空間複雜度: O(1)
        """
        max_sum = nums[0]
        prev_sum = nums[0]
        
        for i in range(1, len(nums)):
            prev_sum = max(nums[i], prev_sum + nums[i])
            max_sum = max(max_sum, prev_sum)
        
        return max_sum
    
    def maxSubArray_divide_conquer(self, nums: List[int]) -> int:
        """
        方法三：分治法
        時間複雜度: O(n log n)
        空間複雜度: O(log n)
        """
        def helper(left, right):
            if left == right:
                return nums[left]
            
            mid = (left + right) // 2
            
            # 左半部分最大和
            left_max = helper(left, mid)
            # 右半部分最大和
            right_max = helper(mid + 1, right)
            # 跨越中點的最大和
            cross_max = cross_sum(left, right, mid)
            
            return max(left_max, right_max, cross_max)
        
        def cross_sum(left, right, mid):
            # 從中點向左的最大和
            left_sum = float('-inf')
            current_sum = 0
            for i in range(mid, left - 1, -1):
                current_sum += nums[i]
                left_sum = max(left_sum, current_sum)
            
            # 從中點向右的最大和
            right_sum = float('-inf')
            current_sum = 0
            for i in range(mid + 1, right + 1):
                current_sum += nums[i]
                right_sum = max(right_sum, current_sum)
            
            return left_sum + right_sum
        
        return helper(0, len(nums) - 1)

class Solution {

     * 方法一：Kadane's Algorithm（推薦）
     * 時間複雜度: O(n)
     * 空間複雜度: O(1)
     */
    int maxSubArray(std::vector& nums) {
        int currentSum = 0;
        int maxSum = INT_MIN;
        
        for (int num : nums) {
            // 決定是繼續累加還是重新開始
            currentSum = std::max(num, currentSum + num);
            // 更新最大值
            maxSum = std::max(maxSum, currentSum);
        }
        
        return maxSum;
    }
    
    /**
     * 方法二：動態規劃
     * 時間複雜度: O(n)
     * 空間複雜度: O(1)
     */
    int maxSubArray_dp(std::vector& nums) {
        int maxSum = nums[0];
        int prevSum = nums[0];
        
        for (int i = 1; i < nums.size(); i++) {
            prevSum = std::max(nums[i], prevSum + nums[i]);
            maxSum = std::max(maxSum, prevSum);
        }
        
        return maxSum;
    }
};

複雜度分析

時間複雜度:
  • Kadane's Algorithm & DP：O(n) - 只需遍歷一次陣列
  • 分治法：O(n log n) - 每層 O(n)，共 log n 層
空間複雜度:
  • Kadane's Algorithm：O(1) - 只用常數個變數
  • DP（優化版）：O(1)
  • 分治法：O(log n) - 遞迴堆疊深度

重點提示

  • Kadane's Algorithm：這是最優解，務必掌握
  • 核心決策：每個位置只需決定「加入前面」還是「重新開始」
  • 為什麼有效：如果前面的累加和是負數，那加上它只會讓結果更小
  • 處理全負數：算法會正確返回最大的負數（最小的虧損）
  • 邊界情況：
    • 只有一個元素 → 返回該元素
    • 全是負數 → 返回最大的負數
    • 全是正數 → 返回陣列總和
  • DP vs Kadane：本質相同，Kadane 是 DP 的空間優化版本
  • 面試技巧：
    • 先說暴力解法 O(n²)
    • 再優化到 Kadane's Algorithm O(n)
    • 可以提到分治法展現進階能力
  • 變形題：
    • 找出具體的子陣列（需要記錄起始和結束位置）
    • Maximum Product Subarray (#152)
    • 環形子陣列最大和
