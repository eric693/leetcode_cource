66. Plus One
Easy

題目描述

給定一個由整數組成的非空陣列所表示的非負整數，在該數的基礎上加一。

最高位數字存放在陣列的首位，陣列中每個元素只存儲單個數字。

你可以假設除了整數 0 之外，這個整數不會以零開頭。

範例 1:
輸入: digits = [1,2,3]
輸出: [1,2,4]
解釋: 陣列表示整數 123，加一後變成 124
範例 2:
輸入: digits = [4,3,2,1]
輸出: [4,3,2,2]
解釋: 陣列表示整數 4321，加一後變成 4322
範例 3:
輸入: digits = [9]
輸出: [1,0]
解釋: 陣列表示整數 9，加一後變成 10
範例 4:
輸入: digits = [9,9,9]
輸出: [1,0,0,0]
解釋: 陣列表示整數 999，加一後變成 1000
限制條件:
1 <= digits.length <= 100
0 <= digits[i] <= 9
digits 不包含任何前導 0

解題思路

方法：從後往前處理進位（推薦）⭐

從最後一位開始加一，處理進位，直到沒有進位或到達陣列開頭。

核心思想：

模擬手算加法，從個位開始加一，如果產生進位則繼續向前處理。

關鍵觀察：

  • 大部分情況下，只需要將最後一位加一
  • 只有遇到 9 時才會產生進位
  • 最壞情況：999...9 → 1000...0（需要在開頭插入 1）

步驟：

  1. 從陣列末尾開始遍歷（從右到左）
  2. 當前位加一：
    • 如果結果 < 10 → 直接返回（無進位）
    • 如果結果 = 10 → 當前位設為 0，繼續向前進位
  3. 如果遍歷完整個陣列還有進位 → 在開頭插入 1

視覺化範例 1：digits = [1,2,3]

初始: [1, 2, 3]
             ↑
           i=2

步驟 1: 3 + 1 = 4 < 10
       無需進位，直接返回
       
結果: [1, 2, 4]

視覺化範例 2：digits = [9,9,9]

初始: [9, 9, 9]
             ↑
           i=2

步驟 1: 9 + 1 = 10
       設置為 0，繼續進位
       [9, 9, 0]
          ↑
        i=1

步驟 2: 9 + 1 = 10
       設置為 0，繼續進位
       [9, 0, 0]
       ↑
     i=0

步驟 3: 9 + 1 = 10
       設置為 0，繼續進位
       [0, 0, 0]
       還有進位！

步驟 4: 在開頭插入 1
       [1, 0, 0, 0]

結果: [1, 0, 0, 0]
優化技巧

提前返回：只要當前位加一後 < 10，就可以直接返回，不需要繼續處理。

這使得大部分情況下（非 9 結尾）只需 O(1) 時間。

程式碼實現

class Solution:
    def plusOne(self, digits: List[int]) -> List[int]:
        """
        方法：從後往前處理進位
        時間複雜度: O(n) - 最壞情況
        空間複雜度: O(1) - 不考慮輸出空間
        """
        n = len(digits)
        
        # 從最後一位開始處理
        for i in range(n - 1, -1, -1):
            # 當前位加一
            digits[i] += 1
            
            # 如果沒有進位，直接返回
            if digits[i] < 10:
                return digits
            
            # 有進位，當前位設為 0
            digits[i] = 0
        
        # 如果執行到這裡，說明所有位都是 9
        # 需要在開頭插入 1
        return [1] + digits
    
    def plusOne_pythonic(self, digits: List[int]) -> List[int]:
        """
        Python 簡潔寫法（轉數字）
        注意：大數可能有精度問題
        """
        num = int(''.join(map(str, digits))) + 1
        return [int(d) for d in str(num)]
#include <vector>
#include <algorithm>

class Solution {
public:
    /**
     * 方法：從後往前處理進位
     * 時間複雜度: O(n)
     * 空間複雜度: O(1)
     */
    vector plusOne(vector& digits) {
        int n = digits.size();
        
        // 從最後一位開始處理
        for (int i = n - 1; i >= 0; i--) {
            // 當前位加一
            digits[i]++;
            
            // 如果沒有進位，直接返回
            if (digits[i] < 10) {
                return digits;
            }
            
            // 有進位，當前位設為 0
            digits[i] = 0;
        }
        
        // 如果執行到這裡，說明所有位都是 9
        // 需要在開頭插入 1
        digits.insert(digits.begin(), 1);
        return digits;
    }
};
class Solution {
    /**
     * 方法：從後往前處理進位
     * 時間複雜度: O(n)
     * 空間複雜度: O(n) - 最壞情況需要新陣列
     */
    public int[] plusOne(int[] digits) {
        int n = digits.length;
        
        // 從最後一位開始處理
        for (int i = n - 1; i >= 0; i--) {
            // 當前位加一
            digits[i]++;
            
            // 如果沒有進位，直接返回
            if (digits[i] < 10) {
                return digits;
            }
            
            // 有進位，當前位設為 0
            digits[i] = 0;
        }
        
        // 如果執行到這裡，說明所有位都是 9
        // 需要創建新陣列，長度 +1
        int[] result = new int[n + 1];
        result[0] = 1;
        // 其餘位默認為 0
        
        return result;
    }
}

複雜度分析

時間複雜度: O(n)

最壞情況下需要遍歷整個陣列（如 999...9），其中 n 是陣列長度。
平均情況：O(1) - 大部分數字最後一位不是 9

空間複雜度: O(1)

不考慮輸出空間，只使用常數個變數。
注意：如果所有位都是 9，需要創建新陣列（Python 用列表拼接，Java 創建新陣列）

重點提示

  • 進位處理：這是經典的陣列進位問題
  • 提前返回：只要當前位 < 10，立即返回
  • 最壞情況：999...9 → 1000...0，需要在開頭加 1
  • 邊界情況：
    • [0] → [1]
    • [9] → [1,0]
    • [9,9,9] → [1,0,0,0]
    • [1,2,3] → [1,2,4]（最常見）
  • 語言差異：
    • Python: 用列表拼接 [1] + digits
    • C++: 用 insert(begin(), 1)
    • Java: 創建新陣列，首位設為 1
  • 常見錯誤：
    • 忘記處理全是 9 的情況
    • 忘記提前返回（無進位時）
    • 進位邏輯錯誤
  • 面試技巧：
    • 先說明從後往前處理進位
    • 舉例說明兩種情況：有進位 vs 無進位
    • 特別說明 999...9 的特殊處理
    • 提到提前返回優化
