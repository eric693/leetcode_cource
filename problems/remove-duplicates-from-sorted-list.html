83. Remove Duplicates from Sorted List
Easy

題目描述

給定一個已排序的鏈表的頭 head，刪除所有重複的元素，使每個元素只出現一次。返回已排序的鏈表。

範例 1:
輸入: head = [1,1,2]
輸出: [1,2]
範例 2:
輸入: head = [1,1,2,3,3]
輸出: [1,2,3]
限制條件:
鏈表中節點數目在範圍 [0, 300] 內
-100 <= Node.val <= 100
題目保證鏈表已經按升序排列

解題思路

核心觀察

由於鏈表已排序，重複元素一定相鄰：

  • 只需比較當前節點和下一個節點
  • 如果值相同，跳過下一個節點
  • 如果不同，繼續前進

關鍵：單指標遍歷，比較相鄰節點

方法：單指標遍歷 ⭐⭐⭐

從頭開始遍歷，刪除重複節點。

算法步驟：

  1. 從頭節點開始
  2. 比較當前節點和下一個節點的值
  3. 如果相同，跳過下一個節點（current.next = current.next.next）
  4. 如果不同，移動到下一個節點
  5. 重複直到鏈表結束

視覺化範例：[1,1,2,3,3]

初始: 1 → 1 → 2 → 3 → 3 → null
      ^
   current

步驟 1: current.val(1) == current.next.val(1)
       刪除下一個
       1 → 2 → 3 → 3 → null
       ^
    current

步驟 2: current.val(1) != current.next.val(2)
       移動指標
       1 → 2 → 3 → 3 → null
           ^
        current

步驟 3: current.val(2) != current.next.val(3)
       移動指標
       1 → 2 → 3 → 3 → null
               ^
            current

步驟 4: current.val(3) == current.next.val(3)
       刪除下一個
       1 → 2 → 3 → null
               ^
            current

步驟 5: current.next == null，結束

最終: 1 → 2 → 3 → null ✓

程式碼實現

# Definition for singly-linked list.
class ListNode:
    def __init__(self, val=0, next=None):
        self.val = val
        self.next = next

class Solution:
    def deleteDuplicates(self, head: ListNode) -> ListNode:
        """
        單指標遍歷
        時間複雜度: O(n)
        空間複雜度: O(1)
        """
        # 處理空鏈表
        if not head:
            return head
        
        current = head
        
        while current and current.next:
            if current.val == current.next.val:
                # 跳過重複節點
                current.next = current.next.next
            else:
                # 移動到下一個節點
                current = current.next
        
        return head
    
    def deleteDuplicates_v2(self, head: ListNode) -> ListNode:
        """
        遞迴版本
        """
        if not head or not head.next:
            return head
        
        head.next = self.deleteDuplicates_v2(head.next)
        
        # 如果當前和下一個值相同，返回下一個
        return head.next if head.val == head.next.val else head
/**
 * Definition for singly-linked list.
 * struct ListNode {
 *     int val;
 *     ListNode *next;
 *     ListNode() : val(0), next(nullptr) {}
 *     ListNode(int x) : val(x), next(nullptr) {}
 *     ListNode(int x, ListNode *next) : val(x), next(next) {}
 * };
 */
class Solution {
public:
    ListNode* deleteDuplicates(ListNode* head) {
        if (!head) return head;
        
        ListNode* current = head;
        
        while (current && current->next) {
            if (current->val == current->next->val) {
                // 跳過重複節點
                ListNode* duplicate = current->next;
                current->next = current->next->next;
                delete duplicate;  // 釋放記憶體
            } else {
                // 移動到下一個節點
                current = current->next;
            }
        }
        
        return head;
    }
};
/**
 * Definition for singly-linked list.
 * public class ListNode {
 *     int val;
 *     ListNode next;
 *     ListNode() {}
 *     ListNode(int val) { this.val = val; }
 *     ListNode(int val, ListNode next) { this.val = val; this.next = next; }
 * }
 */
class Solution {
    public ListNode deleteDuplicates(ListNode head) {
        if (head == null) return head;
        
        ListNode current = head;
        
        while (current != null && current.next != null) {
            if (current.val == current.next.val) {
                // 跳過重複節點
                current.next = current.next.next;
            } else {
                // 移動到下一個節點
                current = current.next;
            }
        }
        
        return head;
    }
}

複雜度分析

時間複雜度: O(n)

遍歷鏈表一次，n 是節點數量

空間複雜度: O(1)

只使用常數個指標

重點提示

  • 已排序的優勢：重複元素必定相鄰
  • 指標操作：current.next = current.next.next
  • 何時移動指標：只有在值不同時才移動
  • 邊界情況：空鏈表、單節點、全部重複
  • 循環條件：while current and current.next
  • 記憶體管理：C++ 需要手動釋放記憶體
  • vs 陣列版本：Remove Duplicates from Sorted Array (#26)
  • 變形題：Remove Duplicates from Sorted List II (#82) - 刪除所有重複
  • 遞迴思路：也可以用遞迴實現
  • 記憶口訣：「相同跳過，不同前進」
