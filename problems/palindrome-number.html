9. Palindrome Number
Easy

題目描述

給定一個整數 x，如果 x 是一個回文整數，返回 true；否則，返回 false。

回文數是指正著讀和反著讀都一樣的整數。例如，121 是回文，而 123 不是。

範例 1:
輸入: x = 121
輸出: true
解釋: 121 從左往右讀是 121，從右往左讀也是 121
範例 2:
輸入: x = -121
輸出: false
解釋: 從左往右讀是 -121，從右往左讀是 121-，因此它不是回文數
範例 3:
輸入: x = 10
輸出: false
解釋: 從右往左讀是 01，因此它不是回文數
限制條件:
-2^31 <= x <= 2^31 - 1
進階:

你能不將整數轉為字串來解決這個問題嗎？

解題思路

方法一：反轉一半數字（推薦）⭐

只反轉數字的後半部分，然後與前半部分比較。

核心思想：

不需要反轉整個數字，只需反轉一半，然後比較前後兩半是否相等。

特殊情況：

  • 負數：一定不是回文（因為負號在前面）
  • 末尾是 0：除了 0 本身，其他都不是回文（如 10, 100）

步驟：

  1. 排除負數和末尾為 0 的數（除了 0）
  2. 反轉後半部分數字
  3. 當反轉的數字 >= 原數字時停止
  4. 比較：
    • 偶數位：x == reversed
    • 奇數位：x == reversed / 10（去掉中間的數字）

視覺化範例：x = 12321

初始: x = 12321, reversed = 0

步驟 1: x=12321, reversed=0
       取出最後一位: 1
       x = 1232, reversed = 1

步驟 2: x=1232, reversed=1
       取出最後一位: 2
       x = 123, reversed = 12

步驟 3: x=123, reversed=12
       取出最後一位: 3
       x = 12, reversed = 123

停止: reversed >= x

比較: x = 12, reversed / 10 = 12
     12 == 12 → true（回文）

範例 2: x = 1221

步驟 1: x = 122, reversed = 1
步驟 2: x = 12, reversed = 12

停止: reversed >= x
比較: x = 12, reversed = 12
     12 == 12 → true（回文）
方法二：轉字串比較（簡單但不推薦）

將數字轉為字串，然後比較字串與其反轉是否相同。

優點：簡單易懂；缺點：使用額外空間，面試官可能不滿意

程式碼實現

class Solution:
    def isPalindrome(self, x: int) -> bool:
        """
        方法一：反轉一半數字（推薦）
        時間複雜度: O(log n)
        空間複雜度: O(1)
        """
        # 負數和末尾為 0 的數（除了 0）不是回文
        if x < 0 or (x % 10 == 0 and x != 0):
            return False
        
        reversed_half = 0
        
        # 反轉後半部分
        while x > reversed_half:
            reversed_half = reversed_half * 10 + x % 10
            x //= 10
        
        # 偶數位：x == reversed_half
        # 奇數位：x == reversed_half // 10（去掉中間數字）
        return x == reversed_half or x == reversed_half // 10
    
    def isPalindrome_string(self, x: int) -> bool:
        """
        方法二：轉字串（不推薦）
        時間複雜度: O(log n)
        空間複雜度: O(log n)
        """
        s = str(x)
        return s == s[::-1]
class Solution {
public:
    /**
     * 方法一：反轉一半數字（推薦）
     * 時間複雜度: O(log n)
     * 空間複雜度: O(1)
     */
    bool isPalindrome(int x) {
        // 負數和末尾為 0 的數（除了 0）不是回文
        if (x < 0 || (x % 10 == 0 && x != 0)) {
            return false;
        }
        
        int reversedHalf = 0;
        
        // 反轉後半部分
        while (x > reversedHalf) {
            reversedHalf = reversedHalf * 10 + x % 10;
            x /= 10;
        }
        
        // 偶數位：x == reversedHalf
        // 奇數位：x == reversedHalf / 10
        return x == reversedHalf || x == reversedHalf / 10;
    }
};

    public boolean isPalindrome(int x) {

複雜度分析

時間複雜度: O(log n)

需要處理數字的一半位數，其中 n 是輸入的整數。

空間複雜度: O(1)

只使用常數個變數。

重點提示

  • 數學方法優於字串：展現對數字運算的理解
  • 只反轉一半：避免整數溢位問題
  • 特殊情況處理：
    • 負數 → false
    • 末尾為 0（除了 0）→ false
    • 0 → true
  • 停止條件：x > reversed（反轉到一半）
  • 奇偶位數：
    • 偶數位（1221）：x == reversed
    • 奇數位（12321）：x == reversed / 10
  • 面試技巧：
    • 先說可以轉字串（展現思路）
    • 立即優化到數學方法（展現能力）
    • 解釋為什麼只需反轉一半
  • 常見錯誤：忘記處理負數和末尾為 0 的情況
