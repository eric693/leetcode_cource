206. Reverse Linked List
Easy

題目描述

給你單向鏈表的頭節點 head，請你反轉鏈表，並返回反轉後的鏈表。

範例 1:
輸入: head = [1,2,3,4,5]
輸出: [5,4,3,2,1]
範例 2:
輸入: head = [1,2]
輸出: [2,1]
範例 3:
輸入: head = []
輸出: []

解題思路

方法一：迭代（推薦）

使用三個指標遍歷鏈表，逐個反轉節點的指向。

核心思想：

遍歷鏈表，將每個節點的 next 指標指向前一個節點，同時保存下一個節點的引用。

步驟：

  1. 初始化三個指標：
    • prev = None（前一個節點）
    • current = head（當前節點）
    • next_node（下一個節點的暫存）
  2. 當 current 不為空時：
    • 保存下一個節點：next_node = current.next
    • 反轉當前節點的指標：current.next = prev
    • 移動指標：prev = current，current = next_node
  3. 返回 prev（新的頭節點）

視覺化範例：head = [1,2,3]

初始: None <- 1 -> 2 -> 3 -> None
      prev   curr

步驟1: None <- 1    2 -> 3 -> None
             prev  curr

步驟2: None <- 1 <- 2    3 -> None
                   prev  curr

步驟3: None <- 1 <- 2 <- 3    None
                        prev  curr

結果: [3,2,1]
方法二：遞迴

使用遞迴的方式反轉鏈表。遞迴到鏈表末尾，然後在回溯過程中反轉指標。

步驟：

  1. 遞迴基礎情況：如果 head 為空或只有一個節點，返回 head
  2. 遞迴反轉後續鏈表，得到新頭節點
  3. 反轉當前節點：head.next.next = head，head.next = None
  4. 返回新頭節點

程式碼實現

# 鏈表節點定義
# class ListNode:
#     def __init__(self, val=0, next=None):
#         self.val = val
#         self.next = next

class Solution:
    def reverseList(self, head: Optional[ListNode]) -> Optional[ListNode]:
        """
        方法一：迭代
        時間複雜度: O(n)
        空間複雜度: O(1)
        """
        prev = None
        current = head
        
        while current:
            # 保存下一個節點
            next_node = current.next
            # 反轉當前節點的指標
            current.next = prev
            # 移動指標
            prev = current
            current = next_node
        
        return prev
    
    def reverseList_recursive(self, head: Optional[ListNode]) -> Optional[ListNode]:
        """
        方法二：遞迴
        時間複雜度: O(n)
        空間複雜度: O(n) - 遞迴堆疊
        """
        # 基礎情況
        if not head or not head.next:
            return head
        
        # 遞迴反轉後續鏈表
        new_head = self.reverseList_recursive(head.next)
        
        # 反轉當前節點
        head.next.next = head
        head.next = None
        
        return new_head
/**
 * 鏈表節點定義
 * struct ListNode {
 *     int val;
 *     ListNode *next;
 *     ListNode() : val(0), next(nullptr) {}
 *     ListNode(int x) : val(x), next(nullptr) {}
 *     ListNode(int x, ListNode *next) : val(x), next(next) {}
 * };
 */

class Solution {
public:
    /**
     * 方法一：迭代
     * 時間複雜度: O(n)
     * 空間複雜度: O(1)
     */
    ListNode* reverseList(ListNode* head) {
        ListNode* prev = nullptr;
        ListNode* current = head;
        
        while (current != nullptr) {
            // 保存下一個節點
            ListNode* nextNode = current->next;
            // 反轉當前節點的指標
            current->next = prev;
            // 移動指標
            prev = current;
            current = nextNode;
        }
        
        return prev;
    }
    
    /**
     * 方法二：遞迴
     * 時間複雜度: O(n)
     * 空間複雜度: O(n)
     */
    ListNode* reverseList_recursive(ListNode* head) {
        // 基礎情況
        if (head == nullptr || head->next == nullptr) {
            return head;
        }
        
        // 遞迴反轉後續鏈表
        ListNode* newHead = reverseList_recursive(head->next);
        
        // 反轉當前節點
        head->next->next = head;
        head->next = nullptr;
        
        return newHead;
    }
};
/**
 * 鏈表節點定義
 * public class ListNode {
 *     int val;
 *     ListNode next;
 *     ListNode() {}
 *     ListNode(int val) { this.val = val; }
 *     ListNode(int val, ListNode next) { this.val = val; this.next = next; }
 * }
 */

class Solution {
    /**
     * 方法一：迭代
     * 時間複雜度: O(n)
     * 空間複雜度: O(1)
     */
    public ListNode reverseList(ListNode head) {
        ListNode prev = null;
        ListNode current = head;
        
        while (current != null) {
            // 保存下一個節點
            ListNode nextNode = current.next;
            // 反轉當前節點的指標
            current.next = prev;
            // 移動指標
            prev = current;
            current = nextNode;
        }
        
        return prev;
    }
    
    /**
     * 方法二：遞迴
     * 時間複雜度: O(n)
     * 空間複雜度: O(n)
     */
    public ListNode reverseList_recursive(ListNode head) {
        // 基礎情況
        if (head == null || head.next == null) {
            return head;
        }
        
        // 遞迴反轉後續鏈表
        ListNode newHead = reverseList_recursive(head.next);
        
        // 反轉當前節點
        head.next.next = head;
        head.next = null;
        
        return newHead;
    }
}

複雜度分析

迭代方法 - 時間複雜度: O(n)

需要遍歷鏈表中的每個節點一次。

迭代方法 - 空間複雜度: O(1)

只使用了常數額外空間。

遞迴方法 - 時間複雜度: O(n)

遍歷所有節點。

遞迴方法 - 空間複雜度: O(n)

遞迴調用堆疊需要 O(n) 空間。

重點提示

  • 迭代 vs 遞迴：迭代方法空間複雜度更優（O(1)），推薦使用
  • 三個指標：prev、current、next_node 是迭代的關鍵
  • 邊界條件：處理空鏈表和單節點鏈表
  • 常見錯誤：忘記保存 next_node，導致鏈表斷裂
  • 應用場景：這是鏈表操作的基礎，很多問題都需要反轉鏈表
