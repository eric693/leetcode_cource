344. Reverse String
Easy

題目描述

編寫一個函數，將輸入的字串反轉。輸入字串以字符陣列 s 的形式給出。

你必須原地修改輸入陣列，使用 O(1) 的額外空間解決這一問題。

範例 1:
輸入: s = ["h","e","l","l","o"]
輸出: ["o","l","l","e","h"]
範例 2:
輸入: s = ["H","a","n","n","a","h"]
輸出: ["h","a","n","n","a","H"]
限制條件:
1 <= s.length <= 10^5
s[i] 是 ASCII 碼表中的可打印字符

解題思路

方法：雙指標交換（推薦）⭐

使用左右雙指標，從兩端向中間移動並交換字符。

核心思想：

最左和最右的字符交換位置，然後指標向中間靠攏，直到相遇。

步驟：

  1. 初始化左指標 left = 0，右指標 right = n-1
  2. 當 left < right 時：
    • 交換 s[left] 和 s[right]
    • left++，right--
  3. 當 left >= right 時停止（已完成反轉）

視覺化範例：s = ["h","e","l","l","o"]

初始: h  e  l  l  o
      ↑           ↑
    left        right

步驟 1: 交換 h 和 o
結果: o  e  l  l  h
         ↑     ↑
       left  right

步驟 2: 交換 e 和 l
結果: o  l  l  e  h
            ↑
         left=right (停止)

最終: ["o","l","l","e","h"]
為什麼是 O(1) 空間？

雖然我們使用了兩個指標變數，但它們占用的空間是常數級別的，與輸入大小無關。交換操作也是原地進行，不需要額外的陣列空間。

程式碼實現

class Solution:
    def reverseString(self, s: List[str]) -> None:
        """
        方法：雙指標交換
        時間複雜度: O(n)
        空間複雜度: O(1)
        
        Do not return anything, modify s in-place instead.
        """
        left = 0
        right = len(s) - 1
        
        while left < right:
            # 交換左右字符
            s[left], s[right] = s[right], s[left]
            # 指標向中間移動
            left += 1
            right -= 1
    
    def reverseString_pythonic(self, s: List[str]) -> None:
        """
        方法：Python 切片（簡潔但可能不符合題意）
        """
        s[:] = s[::-1]
    
    def reverseString_builtin(self, s: List[str]) -> None:
        """
        方法：使用內建函數
        """
        s.reverse()
#include <vector>
#include <algorithm>

class Solution {
public:
    /**
     * 方法：雙指標交換
     * 時間複雜度: O(n)
     * 空間複雜度: O(1)
     */
    void reverseString(std::vector& s) {
        int left = 0;
        int right = s.size() - 1;
        
        while (left < right) {
            // 交換左右字符
            std::swap(s[left], s[right]);
            // 指標向中間移動
            left++;
            right--;
        }
    }
    
    /**
     * 方法：使用標準庫
     */
    void reverseString_stl(std::vector& s) {
        std::reverse(s.begin(), s.end());
    }
};
class Solution {
    /**
     * 方法：雙指標交換
     * 時間複雜度: O(n)
     * 空間複雜度: O(1)
     */
    public void reverseString(char[] s) {
        int left = 0;
        int right = s.length - 1;
        
        while (left < right) {
            // 交換左右字符
            char temp = s[left];
            s[left] = s[right];
            s[right] = temp;
            
            // 指標向中間移動
            left++;
            right--;
        }
    }
}

複雜度分析

時間複雜度: O(n)

需要遍歷陣列的一半，交換 n/2 次，其中 n 是陣列長度。

空間複雜度: O(1)

只使用兩個指標變數和一個臨時變數（用於交換），空間複雜度為常數。

重點提示

  • 原地修改：必須在原陣列上修改，不能創建新陣列
  • 雙指標經典應用：這是學習雙指標的最佳入門題
  • 停止條件：left < right（不是 <=，因為中間元素不需要交換）
  • 交換技巧：
    • Python: a, b = b, a（最簡潔）
    • C++: std::swap(a, b)
    • Java: 需要 temp 變數
  • 邊界情況：
    • 只有一個字符 → 不需要操作
    • 兩個字符 → 交換一次
    • 空陣列 → 題目保證不會出現
  • 面試中：面試官可能要求不使用內建函數（reverse, reversed 等）
  • 擴展：如果要求返回新字串而非原地修改，可以創建新陣列
