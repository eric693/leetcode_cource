38. Count and Say
Easy

題目描述

外觀數列（Look-and-Say sequence）是一個整數序列，從數字 1 開始，序列中的每一項都是對前一項的描述。

前五項如下：

1.     1
2.     11
3.     21
4.     1211
5.     111221

1 被讀作 "one 1" ("一個一") , 即 11。

11 被讀作 "two 1s" ("兩個一"), 即 21。

21 被讀作 "one 2", "one 1" ("一個二", "一個一") , 即 1211。

給定一個正整數 n，輸出外觀數列的第 n 項。

範例 1:
輸入: n = 1
輸出: "1"
解釋: 這是序列的基礎情況
範例 2:
輸入: n = 4
輸出: "1211"
解釋:
countAndSay(1) = "1"
countAndSay(2) = "11" (讀作 "一個一")
countAndSay(3) = "21" (讀作 "兩個一")
countAndSay(4) = "1211" (讀作 "一個二，一個一")
限制條件:
1 <= n <= 30

解題思路

核心觀察

這是一個遞推問題：

  • 每一項都是對前一項的「朗讀」
  • 需要統計連續相同數字的個數
  • 然後說出「個數 + 數字」

關鍵：遍歷字串，統計連續相同字符

方法：迭代生成 ⭐⭐⭐

從 "1" 開始，迭代 n-1 次。

算法步驟：

  1. 從 "1" 開始
  2. 對於每一項，遍歷字串
  3. 統計連續相同字符的個數
  4. 生成 "個數 + 字符" 的描述
  5. 重複 n-1 次

視覺化過程：n = 5

第 1 項: "1"

第 2 項: 讀 "1"
  看到 1 個 '1' → "11"

第 3 項: 讀 "11"
  看到 2 個 '1' → "21"

第 4 項: 讀 "21"
  看到 1 個 '2' → "12"
  看到 1 個 '1' → "11"
  結果: "1211"

第 5 項: 讀 "1211"
  看到 1 個 '1' → "11"
  看到 1 個 '2' → "12"
  看到 2 個 '1' → "21"
  結果: "111221" ✓

統計連續字符的方法：

對於字串 "1211":

i=0: char='1', count=1
     下一個不同 → 說 "11"

i=1: char='2', count=1
     下一個不同 → 說 "12"

i=2: char='1', count=1
i=3: char='1', count=2
     到達末尾 → 說 "21"

最終: "111221"

程式碼實現

class Solution:
    def countAndSay(self, n: int) -> str:
        """
        迭代生成法
        時間複雜度: O(n * m)，m 是字串長度
        空間複雜度: O(m)
        """
        result = "1"
        
        # 迭代 n-1 次
        for _ in range(n - 1):
            result = self.getNext(result)
        
        return result
    
    def getNext(self, s: str) -> str:
        """生成下一項"""
        result = []
        i = 0
        
        while i < len(s):
            char = s[i]
            count = 1
            
            # 統計連續相同字符
            while i + 1 < len(s) and s[i + 1] == char:
                count += 1
                i += 1
            
            # 說出 "個數 + 字符"
            result.append(str(count) + char)
            i += 1
        
        return ''.join(result)
    
    def countAndSay_v2(self, n: int) -> str:
        """
        使用 itertools.groupby
        """
        from itertools import groupby
        
        result = "1"
        
        for _ in range(n - 1):
            result = ''.join(str(len(list(group))) + digit 
                           for digit, group in groupby(result))
        
        return result

using namespace std;

class Solution {
public:
    string countAndSay(int n) {
        string result = "1";
        
        // 迭代 n-1 次
        for (int i = 1; i < n; i++) {
            result = getNext(result);
        }
        
        return result;
    }
    
private:
    string getNext(const string& s) {
        string result = "";
        int i = 0;
        
        while (i < s.length()) {
            char digit = s[i];
            int count = 1;
            
            // 統計連續相同字符
            while (i + 1 < s.length() && s[i + 1] == digit) {
                count++;
                i++;
            }
            
            // 說出 "個數 + 字符"
            result += to_string(count) + digit;
            i++;
        }
        
        return result;
    }
};
class Solution {
    public String countAndSay(int n) {
        String result = "1";
        
        // 迭代 n-1 次
        for (int i = 1; i < n; i++) {
            result = getNext(result);
        }
        
        return result;
    }
    
    private String getNext(String s) {
        StringBuilder result = new StringBuilder();
        int i = 0;
        
        while (i < s.length()) {
            char digit = s.charAt(i);
            int count = 1;
            
            // 統計連續相同字符
            while (i + 1 < s.length() && s.charAt(i + 1) == digit) {
                count++;
                i++;
            }
            
            // 說出 "個數 + 字符"
            result.append(count).append(digit);
            i++;
        }
        
        return result.toString();
    }
}

複雜度分析

時間複雜度: O(n × m)

n 是輸入，m 是字串長度（會隨迭代增長）

空間複雜度: O(m)

每次生成新字串需要的空間

重點提示

  • 理解題意：這是「朗讀」前一項，不是數學計算
  • 統計技巧：使用雙指標統計連續相同字符
  • 順序：先說個數，再說數字
  • Python 技巧：可用 itertools.groupby 簡化
  • 字串增長：長度會隨 n 快速增長
  • 不會出現的數字：永遠不會出現 4 以上的數字
  • Conway 序列：這個序列也被稱為 Conway 序列
  • 有趣性質：數字只會是 1、2、3，不會出現 4 或更大
  • 相關題目：String 操作、遞推生成
  • 記憶口訣：「數個數，說數字」
