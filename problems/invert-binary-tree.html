226. Invert Binary Tree
Easy

題目描述

給定一個二元樹的根節點 root，翻轉這棵二元樹，並返回其根節點。

翻轉的意思是將每個節點的左右子樹交換位置，遞迴地對所有節點進行此操作。

範例 1:
輸入: root = [4,2,7,1,3,6,9]
輸出: [4,7,2,9,6,3,1]

原始樹:          翻轉後:
     4              4
   /   \          /   \
  2     7        7     2
 / \   / \      / \   / \
1   3 6   9    9   6 3   1
範例 2:
輸入: root = [2,1,3]
輸出: [2,3,1]

原始樹:    翻轉後:
  2          2
 / \        / \
1   3      3   1
範例 3:
輸入: root = []
輸出: []

解題思路

方法一：遞迴解法（DFS）

使用深度優先搜尋（DFS）的遞迴方式，對每個節點交換其左右子樹。這是最直觀和簡潔的解法。

核心思想：

對於每個節點，先遞迴翻轉其左右子樹，然後交換左右子節點的位置。

步驟：

  1. 基礎情況：如果節點為空，直接返回 null
  2. 遞迴翻轉左子樹
  3. 遞迴翻轉右子樹
  4. 交換當前節點的左右子節點
  5. 返回當前節點

視覺化範例：[4,2,7,1,3,6,9]

步驟 1: 從根節點開始
     4
   /   \
  2     7
 / \   / \
1   3 6   9

步驟 2: 遞迴處理左子樹 (2)
- 交換 2 的左右子節點: 1 ↔ 3

步驟 3: 遞迴處理右子樹 (7)
- 交換 7 的左右子節點: 6 ↔ 9

步驟 4: 交換根節點的左右子樹
- 交換 4 的左右子節點: 2 ↔ 7

最終結果:
     4
   /   \
  7     2
 / \   / \
9   6 3   1
方法二：迭代解法（BFS）

使用廣度優先搜尋（BFS）配合佇列，逐層處理每個節點並交換其左右子節點。

步驟：

  1. 如果根節點為空，返回 null
  2. 創建一個佇列，將根節點加入佇列
  3. 當佇列不為空時：
    • 取出佇列前端的節點
    • 交換該節點的左右子節點
    • 如果左子節點存在，加入佇列
    • 如果右子節點存在，加入佇列
  4. 返回根節點

程式碼實現

# Definition for a binary tree node.
# class TreeNode:
#     def __init__(self, val=0, left=None, right=None):
#         self.val = val
#         self.left = left
#         self.right = right

class Solution:
    def invertTree(self, root: Optional[TreeNode]) -> Optional[TreeNode]:
        """
        方法一：遞迴解法（DFS）
        時間複雜度: O(n) - 需要訪問每個節點一次
        空間複雜度: O(h) - 遞迴堆疊深度，h 為樹的高度
        """
        # 基礎情況：空節點
        if not root:
            return None
        
        # 遞迴翻轉左右子樹
        left = self.invertTree(root.left)
        right = self.invertTree(root.right)
        
        # 交換左右子節點
        root.left = right
        root.right = left
        
        return root
    
    def invertTree_iterative(self, root: Optional[TreeNode]) -> Optional[TreeNode]:
        """
        方法二：迭代解法（BFS）
        時間複雜度: O(n)
        空間複雜度: O(w) - w 為樹的最大寬度
        """
        if not root:
            return None
        
        from collections import deque
        queue = deque([root])
        
        while queue:
            node = queue.popleft()
            
            # 交換左右子節點
            node.left, node.right = node.right, node.left
            
            # 將子節點加入佇列
            if node.left:
                queue.append(node.left)
            if node.right:
                queue.append(node.right)
        
        return root
#include <queue>

/**
 * Definition for a binary tree node.
 * struct TreeNode {
 *     int val;
 *     TreeNode *left;
 *     TreeNode *right;
 *     TreeNode() : val(0), left(nullptr), right(nullptr) {}
 *     TreeNode(int x) : val(x), left(nullptr), right(nullptr) {}
 *     TreeNode(int x, TreeNode *left, TreeNode *right) : val(x), left(left), right(right) {}
 * };
 */

class Solution {
public:
    /**
     * 方法一：遞迴解法（DFS）
     * 時間複雜度: O(n)
     * 空間複雜度: O(h)
     */
    TreeNode* invertTree(TreeNode* root) {
        // 基礎情況：空節點
        if (root == nullptr) {
            return nullptr;
        }
        
        // 遞迴翻轉左右子樹
        TreeNode* left = invertTree(root->left);
        TreeNode* right = invertTree(root->right);
        
        // 交換左右子節點
        root->left = right;
        root->right = left;
        
        return root;
    }
    
    /**
     * 方法二：迭代解法（BFS）
     * 時間複雜度: O(n)
     * 空間複雜度: O(w)
     */
    TreeNode* invertTree_iterative(TreeNode* root) {
        if (root == nullptr) {
            return nullptr;
        }
        
        std::queue queue;
        queue.push(root);
        
        while (!queue.empty()) {
            TreeNode* node = queue.front();
            queue.pop();
            
            // 交換左右子節點
            TreeNode* temp = node->left;
            node->left = node->right;
            node->right = temp;
            
            // 將子節點加入佇列
            if (node->left != nullptr) {
                queue.push(node->left);
            }
            if (node->right != nullptr) {
                queue.push(node->right);
            }
        }
        
        return root;
    }
};
import java.util.LinkedList;
import java.util.Queue;

/**
 * Definition for a binary tree node.
 * public class TreeNode {
 *     int val;
 *     TreeNode left;
 *     TreeNode right;
 *     TreeNode() {}
 *     TreeNode(int val) { this.val = val; }
 *     TreeNode(int val, TreeNode left, TreeNode right) {
 *         this.val = val;
 *         this.left = left;
 *         this.right = right;
 *     }
 * }
 */

class Solution {
    /**
     * 方法一：遞迴解法（DFS）
     * 時間複雜度: O(n)
     * 空間複雜度: O(h)
     */
    public TreeNode invertTree(TreeNode root) {
        // 基礎情況：空節點
        if (root == null) {
            return null;
        }
        
        // 遞迴翻轉左右子樹
        TreeNode left = invertTree(root.left);
        TreeNode right = invertTree(root.right);
        
        // 交換左右子節點
        root.left = right;
        root.right = left;
        
        return root;
    }
    
    /**
     * 方法二：迭代解法（BFS）
     * 時間複雜度: O(n)
     * 空間複雜度: O(w)
     */
    public TreeNode invertTree_iterative(TreeNode root) {
        if (root == null) {
            return null;
        }
        
        Queue queue = new LinkedList<>();
        queue.offer(root);
        
        while (!queue.isEmpty()) {
            TreeNode node = queue.poll();
            
            // 交換左右子節點
            TreeNode temp = node.left;
            node.left = node.right;
            node.right = temp;
            
            // 將子節點加入佇列
            if (node.left != null) {
                queue.offer(node.left);
            }
            if (node.right != null) {
                queue.offer(node.right);
            }
        }
        
        return root;
    }
}

複雜度分析

時間複雜度: O(n)

需要訪問每個節點恰好一次，其中 n 是二元樹的節點數量。

空間複雜度:
  • 遞迴解法：O(h)，其中 h 是樹的高度。最壞情況下（完全不平衡的樹）h = n，最好情況下（完全平衡的樹）h = log(n)
  • 迭代解法：O(w)，其中 w 是樹的最大寬度。最壞情況下 w = n/2（完全二元樹的最底層）

重點提示

  • 遞迴基礎：這是學習二元樹遞迴的經典入門題目
  • 交換時機：可以在遞迴前或遞迴後交換，結果相同
  • 原地修改：不需要創建新的樹結構，直接修改原樹即可
  • DFS vs BFS：兩種方法都可行，遞迴（DFS）更簡潔，迭代（BFS）更直觀
  • 空節點處理：記得檢查節點是否為 null，避免空指標錯誤
  • 相似題目：對稱二元樹（Symmetric Tree）、鏡像二元樹
