238. Product of Array Except Self
Medium

題目描述

給定一個整數陣列 nums，返回一個陣列 answer，其中 answer[i] 等於 nums 中除了 nums[i] 之外其餘各元素的乘積。

題目數據保證陣列 nums 中任意元素的全部前綴元素和後綴元素的乘積都在 32 位整數範圍內。

⚠️ 重要限制

不能使用除法，並且要求在 O(n) 時間複雜度內完成。

範例 1:
輸入: nums = [1,2,3,4]
輸出: [24,12,8,6]
解釋:
answer[0] = 2 * 3 * 4 = 24
answer[1] = 1 * 3 * 4 = 12
answer[2] = 1 * 2 * 4 = 8
answer[3] = 1 * 2 * 3 = 6
範例 2:
輸入: nums = [-1,1,0,-3,3]
輸出: [0,0,9,0,0]
解釋:
answer[0] = 1 * 0 * (-3) * 3 = 0
answer[1] = (-1) * 0 * (-3) * 3 = 0
answer[2] = (-1) * 1 * (-3) * 3 = 9
answer[3] = (-1) * 1 * 0 * 3 = 0
answer[4] = (-1) * 1 * 0 * (-3) = 0
限制條件:
2 <= nums.length <= 10^5
-30 <= nums[i] <= 30
保證陣列中任意元素的全部前綴元素和後綴元素的乘積都在 32 位整數範圍內
進階:

你能在 O(1) 的額外空間複雜度內完成這個題目嗎？（輸出陣列不被視為額外空間）

解題思路

核心概念：左右累積乘積

對於位置 i，answer[i] = (i 左邊所有元素的乘積) × (i 右邊所有元素的乘積)

視覺化範例：nums = [1, 2, 3, 4]

索引:        0   1   2   3
數值:        1   2   3   4

左側乘積:    1   1   2   6   (不包含自己)
右側乘積:   24  12   4   1   (不包含自己)

結果:       24  12   8   6   (左側 × 右側)
方法一：使用兩個額外陣列

先計算左側乘積陣列，再計算右側乘積陣列，最後相乘。

步驟：

  1. 創建 left 陣列，left[i] = nums[0] * ... * nums[i-1]
  2. 創建 right 陣列，right[i] = nums[i+1] * ... * nums[n-1]
  3. answer[i] = left[i] * right[i]

詳細計算過程：

nums = [1, 2, 3, 4]

計算 left 陣列（從左到右）:
left[0] = 1           (左邊沒有元素)
left[1] = 1           (1)
left[2] = 1 * 2 = 2   (1 * 2)
left[3] = 2 * 3 = 6   (1 * 2 * 3)

計算 right 陣列（從右到左）:
right[3] = 1           (右邊沒有元素)
right[2] = 4           (4)
right[1] = 3 * 4 = 12  (3 * 4)
right[0] = 2*3*4 = 24  (2 * 3 * 4)

計算結果:
answer[0] = left[0] * right[0] = 1 * 24 = 24
answer[1] = left[1] * right[1] = 1 * 12 = 12
answer[2] = left[2] * right[2] = 2 * 4  = 8
answer[3] = left[3] * right[3] = 6 * 1  = 6
方法二：空間優化 O(1)（推薦）⭐

只使用輸出陣列，不需要額外的 left 和 right 陣列。

優化思路：

  1. 第一次遍歷：將左側乘積直接存入 answer
  2. 第二次遍歷：用變數累積右側乘積，並乘到 answer 上

優化過程：

nums = [1, 2, 3, 4]

第一步: 計算左側乘積存入 answer
answer = [1, 1, 2, 6]

第二步: 從右往左，用變數 R 累積右側乘積
R = 1
i=3: answer[3] = 6 * 1 = 6,  R = 1 * 4 = 4
i=2: answer[2] = 2 * 4 = 8,  R = 4 * 3 = 12
i=1: answer[1] = 1 * 12 = 12, R = 12 * 2 = 24
i=0: answer[0] = 1 * 24 = 24

最終: answer = [24, 12, 8, 6]

程式碼實現

class Solution:
    def productExceptSelf(self, nums: List[int]) -> List[int]:
        """
        方法二：空間優化 O(1)（推薦）
        時間複雜度: O(n)
        空間複雜度: O(1) - 不計輸出陣列
        """
        n = len(nums)
        answer = [1] * n
        
        # 第一步：計算左側乘積
        # answer[i] 存儲 nums[0] 到 nums[i-1] 的乘積
        left = 1
        for i in range(n):
            answer[i] = left
            left *= nums[i]
        
        # 第二步：計算右側乘積並乘到 answer 上
        # right 累積 nums[i+1] 到 nums[n-1] 的乘積
        right = 1
        for i in range(n - 1, -1, -1):
            answer[i] *= right
            right *= nums[i]
        
        return answer
    
    def productExceptSelf_twoArrays(self, nums: List[int]) -> List[int]:
        """
        方法一：使用兩個額外陣列
        時間複雜度: O(n)
        空間複雜度: O(n)
        """
        n = len(nums)
        left = [1] * n
        right = [1] * n
        answer = [1] * n
        
        # 計算左側乘積
        for i in range(1, n):
            left[i] = left[i - 1] * nums[i - 1]
        
        # 計算右側乘積
        for i in range(n - 2, -1, -1):
            right[i] = right[i + 1] * nums[i + 1]
        
        # 計算結果
        for i in range(n):
            answer[i] = left[i] * right[i]
        
        return answer

class Solution {
public:
    /**
     * 方法二：空間優化 O(1)（推薦）
     * 時間複雜度: O(n)
     * 空間複雜度: O(1)
     */
    std::vector productExceptSelf(std::vector& nums) {
        int n = nums.size();
        std::vector answer(n, 1);
        
        // 第一步：計算左側乘積
        int left = 1;
        for (int i = 0; i < n; i++) {
            answer[i] = left;
            left *= nums[i];
        }
        
        // 第二步：計算右側乘積並乘到 answer 上
        int right = 1;
        for (int i = n - 1; i >= 0; i--) {
            answer[i] *= right;
            right *= nums[i];
        }
        
        return answer;
    }
    
    /**
     * 方法一：使用兩個額外陣列
     * 時間複雜度: O(n)
     * 空間複雜度: O(n)
     */
    std::vector productExceptSelf_twoArrays(std::vector& nums) {
        int n = nums.size();
        std::vector left(n, 1);
        std::vector right(n, 1);
        std::vector answer(n);
        
        // 計算左側乘積
        for (int i = 1; i < n; i++) {
            left[i] = left[i - 1] * nums[i - 1];
        }
        
        // 計算右側乘積
        for (int i = n - 2; i >= 0; i--) {
            right[i] = right[i + 1] * nums[i + 1];
        }
        
        // 計算結果
        for (int i = 0; i < n; i++) {
            answer[i] = left[i] * right[i];
        }
        
        return answer;
    }
};
class Solution {
    /**
     * 方法二：空間優化 O(1)（推薦）
     * 時間複雜度: O(n)
     * 空間複雜度: O(1)
     */
    public int[] productExceptSelf(int[] nums) {
        int n = nums.length;
        int[] answer = new int[n];
        
        // 第一步：計算左側乘積
        int left = 1;
        for (int i = 0; i < n; i++) {
            answer[i] = left;
            left *= nums[i];
        }
        
        // 第二步：計算右側乘積並乘到 answer 上
        int right = 1;
        for (int i = n - 1; i >= 0; i--) {
            answer[i] *= right;
            right *= nums[i];
        }
        
        return answer;
    }
    
    /**
     * 方法一：使用兩個額外陣列
     * 時間複雜度: O(n)
     * 空間複雜度: O(n)
     */
    public int[] productExceptSelf_twoArrays(int[] nums) {
        int n = nums.length;
        int[] left = new int[n];
        int[] right = new int[n];
        int[] answer = new int[n];
        
        // 初始化
        left[0] = 1;
        right[n - 1] = 1;
        
        // 計算左側乘積
        for (int i = 1; i < n; i++) {
            left[i] = left[i - 1] * nums[i - 1];
        }
        
        // 計算右側乘積
        for (int i = n - 2; i >= 0; i--) {
            right[i] = right[i + 1] * nums[i + 1];
        }
        
        // 計算結果
        for (int i = 0; i < n; i++) {
            answer[i] = left[i] * right[i];
        }
        
        return answer;
    }
}

複雜度分析

方法一：兩個額外陣列
  • 時間複雜度：O(n) - 遍歷三次陣列
  • 空間複雜度：O(n) - left 和 right 陣列
方法二：空間優化（推薦）
  • 時間複雜度：O(n) - 遍歷兩次陣列
  • 空間複雜度：O(1) - 只用常數個變數（不計輸出陣列）

重點提示

  • 為什麼不能用除法？
    • 陣列中可能有 0，除法會出錯
    • 這是題目的核心考點：設計更巧妙的算法
  • 核心技巧：拆分成左右兩部分相乘
  • 空間優化：用變數代替陣列，只遍歷兩次
  • 邊界情況：
    • 第一個元素：左側乘積為 1
    • 最後一個元素：右側乘積為 1
    • 陣列中有 0：不影響算法
  • 面試技巧：
    • 先說明核心思路（左右累積乘積）
    • 畫圖展示計算過程
    • 從 O(n) 空間優化到 O(1)
  • 常見錯誤：
    • 忘記初始化為 1
    • 索引越界
    • 循環方向錯誤
