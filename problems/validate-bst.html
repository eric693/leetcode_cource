98. Validate Binary Search Tree
Medium

題目描述

給定一個二元樹的根節點 root，判斷其是否為有效的二元搜尋樹（BST）。

有效的 BST 定義如下：

  • 節點的左子樹只包含小於當前節點的數
  • 節點的右子樹只包含大於當前節點的數
  • 所有左子樹和右子樹自身也必須是二元搜尋樹
範例 1:
輸入: root = [2,1,3]
輸出: true

解釋:
    2
   / \
  1   3

1 < 2 < 3，滿足 BST 性質
範例 2:
輸入: root = [5,1,4,null,null,3,6]
輸出: false

解釋:
      5
     / \
    1   4
       / \
      3   6

雖然 4 > 5 的右子節點，但 3 < 5，違反了 BST 性質
（右子樹的所有節點都應該 > 5）
範例 3:
輸入: root = [5,4,6,null,null,3,7]
輸出: false

解釋:
      5
     / \
    4   6
       / \
      3   7

3 < 5，但 3 在右子樹中，違反 BST 性質

解題思路

⚠️ 常見錯誤思路

不能只比較節點與其左右子節點的大小！必須確保左子樹的所有節點都小於當前節點，右子樹的所有節點都大於當前節點。

方法一：遞迴 + 範圍限制

對於每個節點，維護一個有效值範圍 [min, max]。遞迴檢查每個節點是否在其有效範圍內。

核心思想：

當我們往左子樹走時，更新上界（max）；往右子樹走時，更新下界（min）。

步驟：

  1. 定義輔助函數 validate(node, min_val, max_val)
    • 檢查節點值是否在 (min_val, max_val) 範圍內
    • 遞迴驗證左子樹：validate(left, min_val, node.val)
    • 遞迴驗證右子樹：validate(right, node.val, max_val)
  2. 初始調用：validate(root, -∞, +∞)

視覺化範例：檢查 [5,1,4,null,null,3,6]

節點 5: 範圍 (-∞, +∞) ✓
        |
    ┌───┴───┐
    ↓       ↓
節點 1:     節點 4:
範圍 (-∞,5) 範圍 (5,+∞)
✓           ✓ (4 不在範圍內!) ✗

當檢查節點 4 時：
- 4 應該 > 5（因為它在 5 的右子樹）
- 但 4 < 5
- 違反 BST 性質，返回 false
方法二：中序遍歷

利用 BST 的性質：中序遍歷（In-order Traversal）會得到一個嚴格遞增的序列。

核心思想：

進行中序遍歷（左 → 根 → 右），同時檢查當前節點是否大於前一個節點。

步驟：

  1. 維護一個變數 prev，記錄中序遍歷的前一個節點值
  2. 遞迴進行中序遍歷：
    • 先遍歷左子樹
    • 檢查當前節點值是否 > prev
    • 更新 prev = 當前節點值
    • 遍歷右子樹
  3. 如果任何時刻發現當前值 ≤ prev，返回 false

中序遍歷範例：

有效的 BST:
    2
   / \
  1   3

中序遍歷: 1 → 2 → 3 (嚴格遞增) ✓

無效的 BST:
      5
     / \
    1   4
       / \
      3   6

中序遍歷: 1 → 5 → 3 → 4 → 6
                5 > 3 (不是遞增!) ✗

程式碼實現

# Definition for a binary tree node.
# class TreeNode:
#     def __init__(self, val=0, left=None, right=None):
#         self.val = val
#         self.left = left
#         self.right = right

class Solution:
    def isValidBST(self, root: Optional[TreeNode]) -> bool:
        """
        方法一：遞迴 + 範圍限制
        時間複雜度: O(n) - 訪問每個節點一次
        空間複雜度: O(h) - 遞迴堆疊深度
        """
        def validate(node, min_val, max_val):
            # 空節點視為有效
            if not node:
                return True
            
            # 檢查當前節點值是否在有效範圍內
            if node.val <= min_val or node.val >= max_val:
                return False
            
            # 遞迴檢查左右子樹
            # 左子樹：所有節點必須 < node.val
            # 右子樹：所有節點必須 > node.val
            return (validate(node.left, min_val, node.val) and
                    validate(node.right, node.val, max_val))
        
        # 初始範圍為 (-∞, +∞)
        return validate(root, float('-inf'), float('inf'))
    
    def isValidBST_inorder(self, root: Optional[TreeNode]) -> bool:
        """
        方法二：中序遍歷
        時間複雜度: O(n)
        空間複雜度: O(h)
        """
        self.prev = float('-inf')
        
        def inorder(node):
            if not node:
                return True
            
            # 遍歷左子樹
            if not inorder(node.left):
                return False
            
            # 檢查當前節點
            if node.val <= self.prev:
                return False
            self.prev = node.val
            
            # 遍歷右子樹
            return inorder(node.right)
        
        return inorder(root)
    
    def isValidBST_iterative(self, root: Optional[TreeNode]) -> bool:
        """
        方法三：迭代中序遍歷
        時間複雜度: O(n)
        空間複雜度: O(h)
        """
        stack = []
        prev = float('-inf')
        current = root
        
        while stack or current:
            # 走到最左邊
            while current:
                stack.append(current)
                current = current.left
            
            # 處理當前節點
            current = stack.pop()
            
            # 檢查是否遞增
            if current.val <= prev:
                return False
            prev = current.val
            
            # 移到右子樹
            current = current.right
        
        return True
#include <stack>
#include <limits>

/**
 * Definition for a binary tree node.
 * struct TreeNode {
 *     int val;
 *     TreeNode *left;
 *     TreeNode *right;
 *     TreeNode() : val(0), left(nullptr), right(nullptr) {}
 *     TreeNode(int x) : val(x), left(nullptr), right(nullptr) {}
 *     TreeNode(int x, TreeNode *left, TreeNode *right) : val(x), left(left), right(right) {}
 * };
 */

class Solution {
public:
    /**
     * 方法一：遞迴 + 範圍限制
     * 時間複雜度: O(n)
     * 空間複雜度: O(h)
     */
    bool isValidBST(TreeNode* root) {
        return validate(root, LLONG_MIN, LLONG_MAX);
    }
    
private:
    bool validate(TreeNode* node, long long minVal, long long maxVal) {
        // 空節點視為有效
        if (node == nullptr) {
            return true;
        }
        
        // 檢查當前節點值是否在有效範圍內
        if (node->val <= minVal || node->val >= maxVal) {
            return false;
        }
        
        // 遞迴檢查左右子樹
        return validate(node->left, minVal, node->val) &&
               validate(node->right, node->val, maxVal);
    }

public:
    /**
     * 方法二：中序遍歷（遞迴）
     * 時間複雜度: O(n)
     * 空間複雜度: O(h)
     */
    bool isValidBST_inorder(TreeNode* root) {
        long long prev = LLONG_MIN;
        return inorder(root, prev);
    }
    
private:
    bool inorder(TreeNode* node, long long& prev) {
        if (node == nullptr) {
            return true;
        }
        
        // 遍歷左子樹
        if (!inorder(node->left, prev)) {
            return false;
        }
        
        // 檢查當前節點
        if (node->val <= prev) {
            return false;
        }
        prev = node->val;
        
        // 遍歷右子樹
        return inorder(node->right, prev);
    }

public:
    /**
     * 方法三：迭代中序遍歷
     * 時間複雜度: O(n)
     * 空間複雜度: O(h)
     */
    bool isValidBST_iterative(TreeNode* root) {
        std::stack stack;
        long long prev = LLONG_MIN;
        TreeNode* current = root;
        
        while (!stack.empty() || current != nullptr) {
            // 走到最左邊
            while (current != nullptr) {
                stack.push(current);
                current = current->left;
            }
            
            // 處理當前節點
            current = stack.top();
            stack.pop();
            
            // 檢查是否遞增
            if (current->val <= prev) {
                return false;
            }
            prev = current->val;
            
            // 移到右子樹
            current = current->right;
        }
        
        return true;
    }
};
import java.util.Stack;

/**
 * Definition for a binary tree node.
 * public class TreeNode {
 *     int val;
 *     TreeNode left;
 *     TreeNode right;
 *     TreeNode() {}
 *     TreeNode(int val) { this.val = val; }
 *     TreeNode(int val, TreeNode left, TreeNode right) {
 *         this.val = val;
 *         this.left = left;
 *         this.right = right;
 *     }
 * }
 */

class Solution {
    /**
     * 方法一：遞迴 + 範圍限制
     * 時間複雜度: O(n)
     * 空間複雜度: O(h)
     */
    public boolean isValidBST(TreeNode root) {
        return validate(root, Long.MIN_VALUE, Long.MAX_VALUE);
    }
    
    private boolean validate(TreeNode node, long minVal, long maxVal) {
        // 空節點視為有效
        if (node == null) {
            return true;
        }
        
        // 檢查當前節點值是否在有效範圍內
        if (node.val <= minVal || node.val >= maxVal) {
            return false;
        }
        
        // 遞迴檢查左右子樹
        return validate(node.left, minVal, node.val) &&
               validate(node.right, node.val, maxVal);
    }
    
    /**
     * 方法二：中序遍歷（遞迴）
     * 時間複雜度: O(n)
     * 空間複雜度: O(h)
     */
    private long prev = Long.MIN_VALUE;
    
    public boolean isValidBST_inorder(TreeNode root) {
        prev = Long.MIN_VALUE;
        return inorder(root);
    }
    
    private boolean inorder(TreeNode node) {
        if (node == null) {
            return true;
        }
        
        // 遍歷左子樹
        if (!inorder(node.left)) {
            return false;
        }
        
        // 檢查當前節點
        if (node.val <= prev) {
            return false;
        }
        prev = node.val;
        
        // 遍歷右子樹
        return inorder(node.right);
    }
    
    /**
     * 方法三：迭代中序遍歷
     * 時間複雜度: O(n)
     * 空間複雜度: O(h)
     */
    public boolean isValidBST_iterative(TreeNode root) {
        Stack stack = new Stack<>();
        long prev = Long.MIN_VALUE;
        TreeNode current = root;
        
        while (!stack.isEmpty() || current != null) {
            // 走到最左邊
            while (current != null) {
                stack.push(current);
                current = current.left;
            }
            
            // 處理當前節點
            current = stack.pop();
            
            // 檢查是否遞增
            if (current.val <= prev) {
                return false;
            }
            prev = current.val;
            
            // 移到右子樹
            current = current.right;
        }
        
        return true;
    }
}

複雜度分析

時間複雜度: O(n)

所有方法都需要訪問每個節點恰好一次，其中 n 是二元樹的節點數量。最壞情況下需要檢查所有節點才能確定是否為有效 BST。

空間複雜度: O(h)

其中 h 是樹的高度。遞迴方法使用函數調用堆疊，迭代方法使用顯式堆疊。

  • 最壞情況（鏈狀樹）：O(n)
  • 最好情況（平衡樹）：O(log n)

重點提示

  • 關鍵陷阱：不能只比較父節點與子節點！必須確保整個左子樹 < 根 < 整個右子樹
  • 範圍限制法：最直觀且容易理解，推薦用於面試
  • 中序遍歷法：利用 BST 的特性，代碼更簡潔
  • 邊界值處理：注意使用 Long.MIN_VALUE/MAX_VALUE 避免整數溢出
  • 等號處理：BST 不允許重複值，所以用 <= 和 >= 而非 < 和 >
  • 測試案例：
    • [2,2,2] → false（不允許重複）
    • [5,1,4,null,null,3,6] → false（右子樹有小於根的值）
    • [Integer.MIN_VALUE] → true（邊界值測試）
