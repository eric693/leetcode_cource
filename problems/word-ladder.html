127. Word Ladder
Hard

題目描述

字典 wordList 中從單詞 beginWord 到 endWord 的轉換序列是一個按下述規格形成的序列：

  • 序列中第一個單詞是 beginWord
  • 序列中最後一個單詞是 endWord
  • 每次轉換只能改變一個字母
  • 轉換過程中的中間單詞必須是字典 wordList 中的單詞

給你兩個單詞 beginWord 和 endWord 和一個字典 wordList，找到從 beginWord 到 endWord 的最短轉換序列中的單詞數量。如果不存在這樣的轉換序列，返回 0。

範例 1:
輸入: beginWord = "hit", endWord = "cog", 
     wordList = ["hot","dot","dog","lot","log","cog"]
輸出: 5
解釋: 一個最短轉換序列是 "hit" -> "hot" -> "dot" -> "dog" -> "cog"，返回它的長度 5。
範例 2:
輸入: beginWord = "hit", endWord = "cog", 
     wordList = ["hot","dot","dog","lot","log"]
輸出: 0
解釋: endWord "cog" 不在字典中，所以無法進行轉換。
限制條件:
1 <= beginWord.length <= 10
endWord.length == beginWord.length
1 <= wordList.length <= 5000
wordList[i].length == beginWord.length
beginWord、endWord 和 wordList[i] 由小寫英文字母組成
beginWord != endWord
wordList 中的所有字串互不相同

解題思路

核心概念：圖的最短路徑 (BFS)

問題建模：

  • 將每個單詞視為圖中的一個節點
  • 如果兩個單詞只差一個字母，則它們之間有一條邊
  • 問題轉化為：從 beginWord 到 endWord 的最短路徑

為什麼用 BFS？

BFS 保證找到的第一條路徑就是最短路徑（無權圖）。

方法一：BFS（推薦）⭐

使用廣度優先搜尋找最短轉換序列。

步驟：

  1. 檢查 endWord 是否在 wordList 中，不在則返回 0
  2. 將 beginWord 加入佇列，標記已訪問
  3. BFS 遍歷：
    • 取出佇列頭部的單詞
    • 嘗試改變每個位置的字母（a-z）
    • 如果新單詞在 wordList 中且未訪問：
      • 如果是 endWord → 返回步數
      • 否則加入佇列繼續搜尋
  4. 如果佇列為空還未找到 → 返回 0

視覺化範例：beginWord = "hit", endWord = "cog"

字典: ["hot","dot","dog","lot","log","cog"]

建立圖（只差一個字母的單詞相連）:
hit → hot
hot → dot, lot
dot → dog, lot
dog → cog, log
lot → log
log → cog

BFS 搜尋過程:
層級 1: hit
層級 2: hot
層級 3: dot, lot
層級 4: dog, log
層級 5: cog ✓ 找到！

最短路徑: hit → hot → dot → dog → cog
長度: 5
優化技巧
  • 使用 Set：將 wordList 轉為 Set，查找 O(1)
  • 雙向 BFS：從起點和終點同時搜尋，速度更快
  • 提前終止：找到 endWord 立即返回

程式碼實現

from collections import deque

class Solution:
    def ladderLength(self, beginWord: str, endWord: str, wordList: List[str]) -> int:
        """
        方法：BFS
        時間複雜度: O(M^2 * N) - M 是單詞長度，N 是單詞數量
        空間複雜度: O(N)
        """
        # 將 wordList 轉為 set，加速查找
        word_set = set(wordList)
        
        # endWord 必須在 wordList 中
        if endWord not in word_set:
            return 0
        
        # BFS 佇列：(當前單詞, 步數)
        queue = deque([(beginWord, 1)])
        visited = {beginWord}
        
        while queue:
            current_word, steps = queue.popleft()
            
            # 嘗試改變每個位置的字母
            for i in range(len(current_word)):
                # 嘗試 a-z
                for c in 'abcdefghijklmnopqrstuvwxyz':
                    # 構造新單詞
                    new_word = current_word[:i] + c + current_word[i+1:]
                    
                    # 找到目標
                    if new_word == endWord:
                        return steps + 1
                    
                    # 新單詞在字典中且未訪問
                    if new_word in word_set and new_word not in visited:
                        visited.add(new_word)
                        queue.append((new_word, steps + 1))
        
        # 無法到達
        return 0
#include <string>
#include <vector>
#include <unordered_set>
#include <queue>

class Solution {
public:
    /**
     * 方法：BFS
     * 時間複雜度: O(M^2 * N)
     * 空間複雜度: O(N)
     */
    int ladderLength(string beginWord, string endWord, vector& wordList) {
        // 將 wordList 轉為 set
        unordered_set word_set(wordList.begin(), wordList.end());
        
        // endWord 必須在 wordList 中
        if (word_set.find(endWord) == word_set.end()) {
            return 0;
        }
        
        // BFS 佇列：(當前單詞, 步數)
        queue> q;
        q.push({beginWord, 1});
        
        unordered_set visited;
        visited.insert(beginWord);
        
        while (!q.empty()) {
            auto [current_word, steps] = q.front();
            q.pop();
            
            // 嘗試改變每個位置的字母
            for (int i = 0; i < current_word.length(); i++) {
                string new_word = current_word;
                
                // 嘗試 a-z
                for (char c = 'a'; c <= 'z'; c++) {
                    new_word[i] = c;
                    
                    // 找到目標
                    if (new_word == endWord) {
                        return steps + 1;
                    }
                    
                    // 新單詞在字典中且未訪問
                    if (word_set.count(new_word) && !visited.count(new_word)) {
                        visited.insert(new_word);
                        q.push({new_word, steps + 1});
                    }
                }
            }
        }
        
        // 無法到達
        return 0;
    }
};
import java.util.*;

class Solution {
    /**
     * 方法：BFS
     * 時間複雜度: O(M^2 * N)
     * 空間複雜度: O(N)
     */
    public int ladderLength(String beginWord, String endWord, List wordList) {
        // 將 wordList 轉為 set
        Set wordSet = new HashSet<>(wordList);
        
        // endWord 必須在 wordList 中
        if (!wordSet.contains(endWord)) {
            return 0;
        }
        
        // BFS 佇列
        Queue> queue = new LinkedList<>();
        queue.offer(new Pair<>(beginWord, 1));
        
        Set visited = new HashSet<>();
        visited.add(beginWord);
        
        while (!queue.isEmpty()) {
            Pair current = queue.poll();
            String currentWord = current.getKey();
            int steps = current.getValue();
            
            // 嘗試改變每個位置的字母
            char[] chars = currentWord.toCharArray();
            for (int i = 0; i < chars.length; i++) {
                char originalChar = chars[i];
                
                // 嘗試 a-z
                for (char c = 'a'; c <= 'z'; c++) {
                    chars[i] = c;
                    String newWord = new String(chars);
                    
                    // 找到目標
                    if (newWord.equals(endWord)) {
                        return steps + 1;
                    }
                    
                    // 新單詞在字典中且未訪問
                    if (wordSet.contains(newWord) && !visited.contains(newWord)) {
                        visited.add(newWord);
                        queue.offer(new Pair<>(newWord, steps + 1));
                    }
                }
                
                // 恢復原字母
                chars[i] = originalChar;
            }
        }
        
        // 無法到達
        return 0;
    }
}

複雜度分析

時間複雜度: O(M² × N)

M 是單詞長度，N 是單詞數量。
對每個單詞（N 個），嘗試改變每個位置（M 個），每次構造新單詞 O(M)。

空間複雜度: O(N)

Set 和 visited 集合最多存儲 N 個單詞，佇列最多存儲 N 個單詞。

重點提示

  • 圖論建模：單詞 → 節點，差一個字母 → 邊
  • BFS 保證最短：第一次到達就是最短路徑
  • Set 加速：將 wordList 轉為 Set，查找 O(1)
  • visited 避免重複：防止環路和重複訪問
  • 邊界情況：
    • endWord 不在字典 → 返回 0
    • beginWord == endWord → 返回 1（題目保證不相等）
    • 字典為空 → 返回 0
  • 優化技巧：
    • 雙向 BFS：從兩端同時搜尋
    • 預處理：建立通用狀態圖（如 h*t, *ot）
  • 面試技巧：
    • 先說明圖論建模思路
    • 解釋為什麼用 BFS 而非 DFS
    • 說明 Set 的作用
