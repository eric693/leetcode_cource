112. Path Sum
Easy

題目描述

給你二元樹的根節點 root 和一個表示目標和的整數 targetSum。判斷該樹中是否存在根節點到葉子節點的路徑，這條路徑上所有節點值相加等於目標和 targetSum。如果存在，返回 true；否則，返回 false。

葉子節點是指沒有子節點的節點。

範例 1:
輸入: root = [5,4,8,11,null,13,4,7,2,null,null,null,1], targetSum = 22
輸出: true
解釋: 路徑 5 → 4 → 11 → 2 的和為 22
        5
       / \
      4   8
     /   / \
    11  13  4
   / \       \
  7   2       1

路徑: 5 → 4 → 11 → 2 = 22 ✓
範例 2:
輸入: root = [1,2,3], targetSum = 5
輸出: false
解釋: 沒有路徑的和等於 5
範例 3:
輸入: root = [], targetSum = 0
輸出: false
解釋: 空樹沒有路徑
限制條件:
樹中節點的數目在範圍 [0, 5000] 內
-1000 <= Node.val <= 1000
-1000 <= targetSum <= 1000

解題思路

核心觀察

關鍵點：

  • 必須是根節點到葉子節點的路徑
  • 不能在中間節點停止
  • 使用遞迴減法：targetSum - node.val

關鍵：遞迴減去當前值，在葉子節點檢查剩餘是否為 0

方法一：遞迴 DFS ⭐⭐⭐

從根節點開始，不斷減去節點值。

算法步驟：

  1. 如果是葉子節點，檢查 targetSum == node.val
  2. 否則遞迴檢查左右子樹
  3. 傳遞 targetSum - node.val 給子樹
  4. 只要有一條路徑滿足即可

視覺化範例：targetSum = 22

        5 (剩餘 22)
       / \
      /   \
     4     8 (剩餘 17)
    /     / \
   11    13  4 (剩餘 14)
  / \         \
 7   2         1 (剩餘 10)

遞迴過程:
1. 根節點 5, 剩餘 = 22 - 5 = 17
   
2. 左子樹: 節點 4, 剩餘 = 17 - 4 = 13
   
3. 左子樹: 節點 11, 剩餘 = 13 - 11 = 2
   
4. 左子樹: 節點 7 (葉子), 剩餘 = 2 - 7 = -5 ✗
   
5. 右子樹: 節點 2 (葉子), 剩餘 = 2 - 2 = 0 ✓

找到路徑! 5 → 4 → 11 → 2 = 22
方法二：迭代（使用棧）⭐⭐

使用棧保存節點和剩餘和。

核心思想：

  • 棧中保存 (節點, 剩餘和) 的配對
  • 遇到葉子節點檢查剩餘和是否為 0
  • 非葉子節點將子節點加入棧

程式碼實現

# Definition for a binary tree node.
class TreeNode:
    def __init__(self, val=0, left=None, right=None):
        self.val = val
        self.left = left
        self.right = right

class Solution:
    def hasPathSum(self, root: TreeNode, targetSum: int) -> bool:
        """
        方法一：遞迴 DFS
        時間複雜度: O(n)
        空間複雜度: O(h)，h 是樹高
        """
        # 空樹
        if not root:
            return False
        
        # 葉子節點：檢查剩餘和是否等於節點值
        if not root.left and not root.right:
            return targetSum == root.val
        
        # 非葉子節點：檢查左右子樹
        # 減去當前節點值後繼續遞迴
        return (self.hasPathSum(root.left, targetSum - root.val) or 
                self.hasPathSum(root.right, targetSum - root.val))
    
    def hasPathSum_v2(self, root: TreeNode, targetSum: int) -> bool:
        """
        方法二：迭代（使用棧）
        時間複雜度: O(n)
        空間複雜度: O(n)
        """
        if not root:
            return False
        
        stack = [(root, targetSum)]
        
        while stack:
            node, remaining = stack.pop()
            
            # 葉子節點
            if not node.left and not node.right:
                if remaining == node.val:
                    return True
            
            # 加入子節點
            if node.right:
                stack.append((node.right, remaining - node.val))
            if node.left:
                stack.append((node.left, remaining - node.val))
        
        return False
/**
 * Definition for a binary tree node.
 * struct TreeNode {
 *     int val;
 *     TreeNode *left;
 *     TreeNode *right;
 *     TreeNode() : val(0), left(nullptr), right(nullptr) {}
 *     TreeNode(int x) : val(x), left(nullptr), right(nullptr) {}
 *     TreeNode(int x, TreeNode *left, TreeNode *right) : val(x), left(left), right(right) {}
 * };
 */
#include 
using namespace std;

class Solution {
public:
    bool hasPathSum(TreeNode* root, int targetSum) {
        if (!root) return false;
        
        // 葉子節點
        if (!root->left && !root->right) {
            return targetSum == root->val;
        }
        
        // 遞迴檢查左右子樹
        return hasPathSum(root->left, targetSum - root->val) || 
               hasPathSum(root->right, targetSum - root->val);
    }
    
    // 迭代版本
    bool hasPathSum_v2(TreeNode* root, int targetSum) {
        if (!root) return false;
        
        stack> stk;
        stk.push({root, targetSum});
        
        while (!stk.empty()) {
            auto [node, remaining] = stk.top();
            stk.pop();
            
            if (!node->left && !node->right) {
                if (remaining == node->val) return true;
            }
            
            if (node->right) 
                stk.push({node->right, remaining - node->val});
            if (node->left) 
                stk.push({node->left, remaining - node->val});
        }
        
        return false;
    }
};
/**
 * Definition for a binary tree node.
 * public class TreeNode {
 *     int val;
 *     TreeNode left;
 *     TreeNode right;
 *     TreeNode() {}
 *     TreeNode(int val) { this.val = val; }
 *     TreeNode(int val, TreeNode left, TreeNode right) {
 *         this.val = val;
 *         this.left = left;
 *         this.right = right;
 *     }
 * }
 */
import java.util.Stack;

class Solution {
    public boolean hasPathSum(TreeNode root, int targetSum) {
        if (root == null) return false;
        
        // 葉子節點
        if (root.left == null && root.right == null) {
            return targetSum == root.val;
        }
        
        // 遞迴檢查左右子樹
        return hasPathSum(root.left, targetSum - root.val) || 
               hasPathSum(root.right, targetSum - root.val);
    }
}

複雜度分析

時間複雜度: O(n)

最壞情況需要遍歷所有節點

空間複雜度: O(h) / O(n)

遞迴：O(h) 棧空間；迭代：O(n) 最壞情況

重點提示

  • 必須到葉子節點：不能在中間節點停止
  • 葉子節點判斷：left == null && right == null
  • 減法技巧：targetSum - node.val 傳給子樹
  • 邊界情況：空樹、單節點、負數路徑
  • 短路求值：使用 or，找到一條即可返回
  • 常見錯誤：在非葉子節點也判斷 sum == 0
  • 擴展問題：Path Sum II (#113) - 返回所有路徑
  • 變形：Path Sum III (#437) - 不必從根開始
  • 相關題目：Binary Tree Paths (#257)
  • 記憶口訣：「減到葉子，檢查是否為零」
