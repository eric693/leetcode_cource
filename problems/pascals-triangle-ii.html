119. Pascal's Triangle II
Easy

題目描述

給定一個非負整數 rowIndex，返回楊輝三角的第 rowIndex 行。

在楊輝三角中，每個數是它左上方和右上方的數的和。

楊輝三角示意圖:
     1
    1 1
   1 2 1
  1 3 3 1
 1 4 6 4 1
範例 1:
輸入: rowIndex = 3
輸出: [1,3,3,1]
範例 2:
輸入: rowIndex = 0
輸出: [1]
範例 3:
輸入: rowIndex = 1
輸出: [1,1]
限制條件:
0 <= rowIndex <= 33
進階:

你可以優化你的算法到 O(rowIndex) 空間複雜度嗎？

解題思路

核心觀察

楊輝三角的性質：

  • 每行首尾都是 1
  • 中間元素 = 上一行對應位置 + 上一行前一個位置
  • 第 n 行有 n+1 個元素
  • row[i] = row[i-1] + row[i]（從右往左更新）

關鍵：使用一個陣列，從右往左原地更新

方法一：逐行生成 ⭐⭐

生成前 rowIndex+1 行，返回最後一行。

缺點：

  • 空間複雜度 O(rowIndex²)
  • 生成了不需要的中間行
方法二：原地更新（最優解）⭐⭐⭐

只使用一個陣列，不斷更新。

核心思想：

  1. 初始化陣列為 [1]
  2. 對於每一行，從右往左更新
  3. row[j] = row[j] + row[j-1]
  4. 最後在末尾加 1

為什麼從右往左？

錯誤：從左往右
當前: [1, 2, 1]
更新 row[1] = row[1] + row[0] = 2 + 1 = 3
      [1, 3, 1]  ← row[0] 已被覆蓋！
更新 row[2] = row[2] + row[1] = 1 + 3 = 4 ✗
      [1, 3, 4]  錯誤！

正確：從右往左
當前: [1, 2, 1]
更新 row[2] = row[2] + row[1] = 1 + 2 = 3
      [1, 2, 3]
更新 row[1] = row[1] + row[0] = 2 + 1 = 3
      [1, 3, 3]
加入 1: [1, 3, 3, 1] ✓

完整過程：生成第 4 行

初始: [1]

第 1 行:
  [1] + 1 = [1, 1]

第 2 行:
  從右往左: [1, 1]
  row[1] = row[1] + row[0] = 1 + 1 = 2
  [1, 2] + 1 = [1, 2, 1]

第 3 行:
  從右往左: [1, 2, 1]
  row[2] = row[2] + row[1] = 1 + 2 = 3
  row[1] = row[1] + row[0] = 2 + 1 = 3
  [1, 3, 3] + 1 = [1, 3, 3, 1]

第 4 行:
  從右往左: [1, 3, 3, 1]
  row[3] = row[3] + row[2] = 1 + 3 = 4
  row[2] = row[2] + row[1] = 3 + 3 = 6
  row[1] = row[1] + row[0] = 3 + 1 = 4
  [1, 4, 6, 4] + 1 = [1, 4, 6, 4, 1] ✓

程式碼實現

class Solution:
    def getRow(self, rowIndex: int) -> List[int]:
        """
        方法：原地更新（最優解）
        時間複雜度: O(rowIndex²)
        空間複雜度: O(1)，不計算輸出空間
        """
        row = [1]
        
        for i in range(rowIndex):
            # 從右往左更新
            for j in range(len(row) - 1, 0, -1):
                row[j] = row[j] + row[j - 1]
            # 在末尾加 1
            row.append(1)
        
        return row
    
    def getRow_v2(self, rowIndex: int) -> List[int]:
        """
        使用組合數公式
        C(n, k) = n! / (k! * (n-k)!)
        C(n, k) = C(n, k-1) * (n-k+1) / k
        """
        row = [1]
        for k in range(1, rowIndex + 1):
            # C(rowIndex, k) = C(rowIndex, k-1) * (rowIndex - k + 1) / k
            row.append(row[-1] * (rowIndex - k + 1) // k)
        return row
#include <vector>
using namespace std;

class Solution {
public:
    vector getRow(int rowIndex) {
        vector row = {1};
        
        for (int i = 0; i < rowIndex; i++) {
            // 從右往左更新
            for (int j = row.size() - 1; j > 0; j--) {
                row[j] = row[j] + row[j - 1];
            }
            // 在末尾加 1
            row.push_back(1);
        }
        
        return row;
    }
};
import java.util.ArrayList;
import java.util.List;

class Solution {
    public List getRow(int rowIndex) {
        List row = new ArrayList<>();
        row.add(1);
        
        for (int i = 0; i < rowIndex; i++) {
            // 從右往左更新
            for (int j = row.size() - 1; j > 0; j--) {
                row.set(j, row.get(j) + row.get(j - 1));
            }
            // 在末尾加 1
            row.add(1);
        }
        
        return row;
    }
}

複雜度分析

時間複雜度: O(rowIndex²)

外層循環 rowIndex 次，內層循環平均 rowIndex/2 次

空間複雜度: O(1)

只使用一個陣列，不計算輸出空間

重點提示

  • 從右往左：避免覆蓋需要的舊值
  • 更新公式：row[j] = row[j] + row[j-1]
  • 索引範圍：j 從 len(row)-1 到 1
  • 每行末尾：別忘記加 1
  • 初始化：[1] 代表第 0 行
  • 組合數公式：C(n,k) = C(n,k-1) * (n-k+1) / k
  • vs Pascal's Triangle I：#118 返回所有行
  • 數學性質：第 n 行是二項式係數
  • 對稱性：每行是對稱的
  • 記憶口訣：「從右往左，加左成新」
