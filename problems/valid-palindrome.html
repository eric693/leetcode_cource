125. Valid Palindrome
Easy

題目描述

給定一個字串 s，判斷它是否為回文字串（palindrome）。只考慮字母和數字字符，忽略字母的大小寫。

回文字串是指正著讀和反著讀都一樣的字串。

範例 1:
輸入: s = "A man, a plan, a canal: Panama"
輸出: true
解釋: "amanaplanacanalpanama" 是回文
範例 2:
輸入: s = "race a car"
輸出: false
解釋: "raceacar" 不是回文
範例 3:
輸入: s = " "
輸出: true
解釋: 空字串（移除非字母數字字符後）被視為回文

解題思路

方法：雙指標（Two Pointers）

使用兩個指標，一個從字串開頭開始，一個從字串結尾開始，向中間移動並比較字符。

核心思想：

回文字串的特性是從兩端往中間讀取時，對應位置的字符應該相同。使用雙指標可以在 O(n) 時間內完成檢查。

步驟：

  1. 初始化兩個指標：left 指向字串開頭，right 指向字串結尾
  2. 當 left < right 時，循環執行：
    • 如果 s[left] 不是字母或數字，left 向右移動
    • 如果 s[right] 不是字母或數字，right 向左移動
    • 比較 s[left] 和 s[right]（忽略大小寫）：
      • 如果不相同，返回 false
      • 如果相同，left 右移，right 左移
  3. 如果循環正常結束，返回 true

為什麼使用雙指標：

  • 只需遍歷字串一次，不需要額外建立新字串
  • 空間複雜度為 O(1)
  • 可以提前終止（遇到不匹配時）

程式碼實現

class Solution:
    def isPalindrome(self, s: str) -> bool:
        """
        使用雙指標檢查回文
        時間複雜度: O(n)
        空間複雜度: O(1)
        """
        # 初始化左右指標
        left, right = 0, len(s) - 1
        
        while left < right:
            # 跳過非字母數字字符（左指標）
            while left < right and not s[left].isalnum():
                left += 1
            
            # 跳過非字母數字字符（右指標）
            while left < right and not s[right].isalnum():
                right -= 1
            
            # 比較字符（忽略大小寫）
            if s[left].lower() != s[right].lower():
                return False
            
            # 移動指標
            left += 1
            right -= 1
        
        return True
#include <string>
#include <cctype>

class Solution {
public:
    /**
     * 使用雙指標檢查回文
     * 時間複雜度: O(n)
     * 空間複雜度: O(1)
     */
    bool isPalindrome(std::string s) {
        int left = 0, right = s.length() - 1;
        
        while (left < right) {
            // 跳過非字母數字字符（左指標）
            while (left < right && !isalnum(s[left])) {
                left++;
            }
            
            // 跳過非字母數字字符（右指標）
            while (left < right && !isalnum(s[right])) {
                right--;
            }
            
            // 比較字符（轉換為小寫）
            if (tolower(s[left]) != tolower(s[right])) {
                return false;
            }
            
            // 移動指標
            left++;
            right--;
        }
        
        return true;
    }
};
class Solution {
    /**
     * 使用雙指標檢查回文
     * 時間複雜度: O(n)
     * 空間複雜度: O(1)
     */
    public boolean isPalindrome(String s) {
        int left = 0, right = s.length() - 1;
        
        while (left < right) {
            // 跳過非字母數字字符（左指標）
            while (left < right && !Character.isLetterOrDigit(s.charAt(left))) {
                left++;
            }
            
            // 跳過非字母數字字符（右指標）
            while (left < right && !Character.isLetterOrDigit(s.charAt(right))) {
                right--;
            }
            
            // 比較字符（轉換為小寫）
            if (Character.toLowerCase(s.charAt(left)) != 
                Character.toLowerCase(s.charAt(right))) {
                return false;
            }
            
            // 移動指標
            left++;
            right--;
        }
        
        return true;
    }
}

複雜度分析

時間複雜度: O(n)

每個字符最多被訪問一次，其中 n 是字串的長度。

空間複雜度: O(1)

只使用了常數額外空間（兩個指標變數）。

重點提示

  • 雙指標模式：這是處理回文問題的經典模式，適用於陣列和字串
  • 字符判斷：Python 使用 isalnum()，C++ 使用 isalnum()，Java 使用 Character.isLetterOrDigit()
  • 大小寫轉換：統一轉換為小寫進行比較，避免大小寫差異
  • 邊界條件：注意處理空字串和只有非字母數字字符的情況
  • 提前終止：一旦發現不匹配就可以立即返回 false，不需要繼續檢查
