79. Word Search
Medium

題目描述

給定一個 m x n 二維字符網格 board 和一個字串 word。如果 word 存在於網格中，返回 true；否則，返回 false。

單詞必須按照字母順序，通過相鄰的單元格內的字母構成，其中「相鄰」單元格是那些水平相鄰或垂直相鄰的單元格。同一個單元格內的字母不允許被重複使用。

範例 1:
輸入: board = [["A","B","C","E"],
              ["S","F","C","S"],
              ["A","D","E","E"]], 
     word = "ABCCED"
輸出: true

視覺化:
A → B → C ↓
S   F   C   E
A   D   E ← E
範例 2:
輸入: board = [["A","B","C","E"],
              ["S","F","C","S"],
              ["A","D","E","E"]], 
     word = "SEE"
輸出: true

視覺化:
A   B   C   E
S → E ← E   S
A   D   E   E
    ↓
    (S-E-E)
範例 3:
輸入: board = [["A","B","C","E"],
              ["S","F","C","S"],
              ["A","D","E","E"]], 
     word = "ABCB"
輸出: false
解釋: 無法找到，因為同一個格子不能重複使用
限制條件:
m == board.length
n = board[i].length
1 <= m, n <= 6
1 <= word.length <= 15
board 和 word 僅由大小寫英文字母組成

解題思路

核心觀察

這是一個典型的回溯（Backtracking）問題：

  • 需要在二維網格中搜尋路徑
  • 可以上下左右移動
  • 每個格子只能使用一次
  • 需要嘗試所有可能的路徑

關鍵：DFS + 回溯 + 訪問標記

方法：DFS + 回溯 ⭐⭐⭐

使用深度優先搜尋和回溯來探索所有可能的路徑。

核心思想：

  • 從每個格子作為起點嘗試
  • DFS 探索四個方向
  • 匹配成功繼續，失敗則回溯
  • 使用標記避免重複訪問

算法步驟：

  1. 遍歷 board 的每個格子作為起點
  2. 從起點開始 DFS：
    • 檢查當前字符是否匹配
    • 標記當前格子為已訪問
    • 遞迴探索四個方向
    • 回溯：恢復標記
  3. 如果找到完整單詞，返回 true

視覺化範例：word = "ABCCED"

Board:
A  B  C  E
S  F  C  S
A  D  E  E

搜尋過程（從 A 開始）:

步驟 1: 找到 'A' at (0,0)
[A] B  C  E
 S  F  C  S
 A  D  E  E
匹配: A ✓

步驟 2: 向右找 'B' at (0,1)
[A][B] C  E
 S  F  C  S
 A  D  E  E
匹配: AB ✓

步驟 3: 向右找 'C' at (0,2)
[A][B][C] E
 S  F  C  S
 A  D  E  E
匹配: ABC ✓

步驟 4: 向下找 'C' at (1,2)
[A][B][C] E
 S  F [C] S
 A  D  E  E
匹配: ABCC ✓

步驟 5: 向下找 'E' at (2,2)
[A][B][C] E
 S  F [C] S
 A  D [E] E
匹配: ABCCE ✓

步驟 6: 向右找 'D'？不匹配，回溯
向左找 'D'？不匹配
向上找 'D'？不匹配
向下找 'D'？越界

回溯到步驟 4，嘗試其他方向...

最終找到正確路徑:
A → B → C
        ↓
        C
        ↓
        E → D (不對)

重新嘗試:
A → B → C
        ↓
        C → S? (不對)

繼續回溯探索...

訪問標記的方法：

  • 方法 1：使用 visited 陣列
  • 方法 2：修改 board（臨時改成特殊字符）
  • 方法 3：使用 set 記錄座標

剪枝優化：

  • 邊界檢查
  • 字符不匹配立即返回
  • 已訪問過立即返回
  • 提前找到答案立即返回

程式碼實現

class Solution:
    def exist(self, board: List[List[str]], word: str) -> bool:
        """
        DFS + 回溯
        時間複雜度: O(m×n×4^L)，L 是單詞長度
        空間複雜度: O(L)，遞迴棧深度
        """
        if not board or not board[0]:
            return False
        
        m, n = len(board), len(board[0])
        
        def dfs(i, j, index):
            """
            從 (i,j) 開始搜尋 word[index:]
            """
            # 找到完整單詞
            if index == len(word):
                return True
            
            # 邊界檢查
            if i < 0 or i >= m or j < 0 or j >= n:
                return False
            
            # 字符不匹配或已訪問
            if board[i][j] != word[index]:
                return False
            
            # 標記為已訪問
            temp = board[i][j]
            board[i][j] = '#'
            
            # 探索四個方向
            found = (dfs(i+1, j, index+1) or  # 下
                    dfs(i-1, j, index+1) or  # 上
                    dfs(i, j+1, index+1) or  # 右
                    dfs(i, j-1, index+1))    # 左
            
            # 回溯：恢復訪問標記
            board[i][j] = temp
            
            return found
        
        # 從每個格子作為起點嘗試
        for i in range(m):
            for j in range(n):
                if dfs(i, j, 0):
                    return True
        
        return False
    
    def exist_v2(self, board: List[List[str]], word: str) -> bool:
        """
        使用 visited 集合的版本
        """
        m, n = len(board), len(board[0])
        
        def dfs(i, j, index, visited):
            if index == len(word):
                return True
            
            if (i < 0 or i >= m or j < 0 or j >= n or
                (i, j) in visited or board[i][j] != word[index]):
                return False
            
            visited.add((i, j))
            
            found = (dfs(i+1, j, index+1, visited) or
                    dfs(i-1, j, index+1, visited) or
                    dfs(i, j+1, index+1, visited) or
                    dfs(i, j-1, index+1, visited))
            
            visited.remove((i, j))
            
            return found
        
        for i in range(m):
            for j in range(n):
                if dfs(i, j, 0, set()):
                    return True
        
        return False
#include <vector>
#include <string>
using namespace std;

class Solution {
public:
    /**
     * DFS + 回溯
     * 時間複雜度: O(m×n×4^L)
     * 空間複雜度: O(L)
     */
    bool exist(vector>& board, string word) {
        if (board.empty() || board[0].empty()) return false;
        
        m = board.size();
        n = board[0].size();
        
        for (int i = 0; i < m; i++) {
            for (int j = 0; j < n; j++) {
                if (dfs(board, word, i, j, 0)) {
                    return true;
                }
            }
        }
        
        return false;
    }
    
private:
    int m, n;
    
    bool dfs(vector>& board, string& word, int i, int j, int index) {
        if (index == word.length()) {
            return true;
        }
        
        if (i < 0 || i >= m || j < 0 || j >= n || board[i][j] != word[index]) {
            return false;
        }
        
        char temp = board[i][j];
        board[i][j] = '#';
        
        bool found = dfs(board, word, i+1, j, index+1) ||
                     dfs(board, word, i-1, j, index+1) ||
                     dfs(board, word, i, j+1, index+1) ||
                     dfs(board, word, i, j-1, index+1);
        
        board[i][j] = temp;
        
        return found;
    }
};
class Solution {
    private int m, n;
    
    /**
     * DFS + 回溯
     * 時間複雜度: O(m×n×4^L)
     * 空間複雜度: O(L)
     */
    public boolean exist(char[][] board, String word) {
        if (board == null || board.length == 0 || board[0].length == 0) {
            return false;
        }
        
        m = board.length;
        n = board[0].length;
        
        for (int i = 0; i < m; i++) {
            for (int j = 0; j < n; j++) {
                if (dfs(board, word, i, j, 0)) {
                    return true;
                }
            }
        }
        
        return false;
    }
    
    private boolean dfs(char[][] board, String word, int i, int j, int index) {
        if (index == word.length()) {
            return true;
        }
        
        if (i < 0 || i >= m || j < 0 || j >= n || board[i][j] != word.charAt(index)) {
            return false;
        }
        
        char temp = board[i][j];
        board[i][j] = '#';
        
        boolean found = dfs(board, word, i+1, j, index+1) ||
                        dfs(board, word, i-1, j, index+1) ||
                        dfs(board, word, i, j+1, index+1) ||
                        dfs(board, word, i, j-1, index+1);
        
        board[i][j] = temp;
        
        return found;
    }
}

複雜度分析

時間複雜度: O(m×n×4^L)

m×n 個起點，每個位置最多探索 4 個方向，單詞長度 L。實際會因剪枝更快。

空間複雜度: O(L)

遞迴棧深度等於單詞長度。

重點提示

  • 回溯三要素：
    • 選擇：四個方向
    • 約束：邊界、字符匹配、未訪問
    • 目標：找到完整單詞
  • 訪問標記技巧：
    • 臨時修改 board 為特殊字符（'#'）
    • 回溯時恢復原值
    • 避免額外空間
  • DFS 順序：
    • 上下左右任意順序都可以
    • 使用 or 短路求值
    • 找到答案立即返回
  • 剪枝優化：
    • 提前檢查邊界
    • 字符不匹配立即返回
    • 找到答案不再探索
  • 常見錯誤：
    • 忘記回溯恢復標記
    • 邊界條件檢查不全
    • 忘記檢查已訪問
  • 面試技巧：回溯模板的典型應用
