122. Best Time to Buy and Sell Stock II
Medium

題目描述

給定一個陣列 prices，其中 prices[i] 表示某支股票第 i 天的價格。

設計一個算法來計算你所能獲取的最大利潤。你可以完成任意次數的交易（即買入和賣出一支股票）。

注意：你不能同時參與多筆交易（你必須在再次購買前出售掉之前的股票）。

範例 1:
輸入: prices = [7,1,5,3,6,4]
輸出: 7
解釋: 在第 2 天（股票價格 = 1）買入，在第 3 天（股票價格 = 5）賣出，利潤 = 5-1 = 4。
     然後在第 4 天（股票價格 = 3）買入，在第 5 天（股票價格 = 6）賣出，利潤 = 6-3 = 3。
     總利潤 = 4 + 3 = 7。
範例 2:
輸入: prices = [1,2,3,4,5]
輸出: 4
解釋: 在第 1 天（股票價格 = 1）買入，在第 5 天（股票價格 = 5）賣出，利潤 = 5-1 = 4。
     注意你不能在第 1 天和第 2 天接連購買股票，之後再將它們賣出。
     因為這樣屬於同時參與了多筆交易，你必須在再次購買前出售掉之前的股票。
範例 3:
輸入: prices = [7,6,4,3,1]
輸出: 0
解釋: 在這種情況下，沒有交易完成，最大利潤為 0。
限制條件:
1 <= prices.length <= 3 * 10^4
0 <= prices[i] <= 10^4

解題思路

方法：貪心算法（推薦）⭐

抓住每一次價格上漲的機會，累加所有正向差價。

核心思想：

只要明天的價格比今天高，就在今天買入明天賣出。把所有上漲的利潤都抓住。

關鍵洞察：

  • 如果價格持續上漲（如 1→2→3→4），可以分解為多次交易：
    • 方案 A：第 1 天買入，第 4 天賣出 → 利潤 = 4-1 = 3
    • 方案 B：(2-1) + (3-2) + (4-3) = 1+1+1 = 3 ✓
    • 兩種方案利潤相同！所以只需累加每次上漲的差價

步驟：

  1. 遍歷價格陣列（從第 2 天開始）
  2. 如果今天的價格 > 昨天的價格：
    • 利潤 += 今天價格 - 昨天價格
  3. 返回總利潤

視覺化範例：prices = [7,1,5,3,6,4]

價格:  7   1   5   3   6   4
        ↓   ↓   ↑   ↓   ↑   ↓
差價:     -6  +4  -2  +3  -2

策略: 只抓上漲
     第 1→2 天: 1-7 = -6  ✗ 下跌，不交易
     第 2→3 天: 5-1 = +4  ✓ 上漲，獲利 4
     第 3→4 天: 3-5 = -2  ✗ 下跌，不交易
     第 4→5 天: 6-3 = +3  ✓ 上漲，獲利 3
     第 5→6 天: 4-6 = -2  ✗ 下跌，不交易

總利潤: 4 + 3 = 7

連續上漲範例：prices = [1,2,3,4,5]

價格:  1   2   3   4   5
        ↑   ↑   ↑   ↑   ↑
差價:     +1  +1  +1  +1

策略: 每天都上漲，每天都交易
     1→2: +1
     2→3: +1
     3→4: +1
     4→5: +1

總利潤: 1+1+1+1 = 4
等同於: 5-1 = 4 ✓
為什麼貪心算法有效？

數學證明：

假設價格序列為 a₁, a₂, ..., aₙ

最優策略的總利潤 = Σ max(0, aᵢ₊₁ - aᵢ)

即：累加所有正向差價

直覺理解：

  • 持續上漲的區間，無論一次買賣還是多次買賣，總利潤相同
  • 下跌時不交易，利潤為 0
  • 所以：抓住每一次上漲機會 = 最大利潤

程式碼實現

class Solution:
    def maxProfit(self, prices: List[int]) -> int:
        """
        方法：貪心算法
        時間複雜度: O(n)
        空間複雜度: O(1)
        """
        profit = 0
        
        # 從第二天開始遍歷
        for i in range(1, len(prices)):
            # 如果今天價格比昨天高，就交易
            if prices[i] > prices[i - 1]:
                profit += prices[i] - prices[i - 1]
        
        return profit
    
    def maxProfit_pythonic(self, prices: List[int]) -> int:
        """
        Python 簡潔寫法
        """
        return sum(max(0, prices[i] - prices[i-1]) 
                   for i in range(1, len(prices)))

複雜度分析

時間複雜度: O(n)

只需遍歷價格陣列一次，其中 n 是天數。

空間複雜度: O(1)

只使用常數個變數。

重點提示

  • 貪心思想：抓住每一次上漲的機會
  • 核心公式：profit += max(0, prices[i] - prices[i-1])
  • 為什麼有效：連續上漲拆分成多次交易，利潤不變
  • 與 Stock I 對比：
    • Stock I (#121): 只能交易一次 → 找最大差價
    • Stock II (#122): 可以多次交易 → 累加所有上漲 ⭐
  • 邊界情況：
    • 只有一天 → 無法交易，利潤 0
    • 持續下跌 → 不交易，利潤 0
    • 持續上漲 → 每天都交易
  • 常見誤解：
    • ❌ 誤：需要找到最優買賣時機
    • ✓ 正：只要上漲就立即交易
  • 面試技巧：
    • 先說明貪心思想
    • 用連續上漲的例子證明有效性
    • 對比 Stock I 展現理解深度
    • 說明時間和空間都是最優的
