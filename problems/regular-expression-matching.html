10. Regular Expression Matching
Hard

題目描述

給定一個字串 s 和一個字元模式 p，實現一個支持 '.' 和 '*' 的正則表達式匹配。

  • '.' 匹配任意單個字元
  • '*' 匹配零個或多個前面的那一個元素

所謂匹配，是要涵蓋整個字串 s 的，而不是部分字串。

範例 1:
輸入: s = "aa", p = "a"
輸出: false
解釋: "a" 無法匹配 "aa" 整個字串
範例 2:
輸入: s = "aa", p = "a*"
輸出: true
解釋: '*' 代表可以匹配零個或多個前面的那一個元素，即 'a'。
     因此重複 'a' 一次，字串就是 "aa"。
範例 3:
輸入: s = "ab", p = ".*"
輸出: true
解釋: ".*" 表示可匹配零個或多個('*')任意字元('.')。
範例 4:
輸入: s = "aab", p = "c*a*b"
輸出: true
解釋: c 可以不被重複，a 可以被重複 2 次。因此可以匹配字串 "aab"。
範例 5:
輸入: s = "mississippi", p = "mis*is*p*."
輸出: false
限制條件:
1 <= s.length <= 20
1 <= p.length <= 20
s 只包含從 a-z 的小寫字母
p 只包含從 a-z 的小寫字母，以及字元 . 和 *
保證每次出現字元 * 時，前面都匹配到有效的字元

解題思路

核心概念：動態規劃

為什麼用 DP？

匹配問題有最優子結構：整個字串的匹配可以由子字串的匹配決定。

狀態定義：

dp[i][j] 表示 s 的前 i 個字元與 p 的前 j 個字元是否匹配

關鍵規則：

  • 普通字元：s[i-1] == p[j-1] → dp[i][j] = dp[i-1][j-1]
  • '.' 字元：匹配任意字元 → dp[i][j] = dp[i-1][j-1]
  • '*' 字元：複雜情況（見下方詳解）
處理 '*' 的邏輯

'*' 可以讓前一個字元重複 0 次或多次。

兩種情況：

  1. 匹配 0 次（忽略 x*）：
    • dp[i][j] = dp[i][j-2]
    • 例如："ab" 匹配 "abc*" → 忽略 c*
  2. 匹配 1 次或多次：
    • 前提：s[i-1] 與 p[j-2] 匹配（p[j-2] 是 * 前面的字元）
    • dp[i][j] = dp[i-1][j]
    • 例如："aaa" 匹配 "a*" → 繼續用 a* 匹配前面的字元

DP 表格範例：s = "aab", p = "c*a*b"

"" c c* c*a c*a* c*a*b
"" T F T F T F
a F F F T T F
aa F F F F T F
aab F F F F F T

dp[3][6] = T，表示 "aab" 可以匹配 "c*a*b"

狀態轉移方程
如果 p[j-1] != '*':
    if s[i-1] == p[j-1] or p[j-1] == '.':
        dp[i][j] = dp[i-1][j-1]
    else:
        dp[i][j] = false

如果 p[j-1] == '*':
    # 情況 1: 匹配 0 次（忽略 x*）
    dp[i][j] = dp[i][j-2]
    
    # 情況 2: 匹配 1+ 次
    if s[i-1] == p[j-2] or p[j-2] == '.':
        dp[i][j] = dp[i][j] or dp[i-1][j]

程式碼實現

class Solution:
    def isMatch(self, s: str, p: str) -> bool:
        """
        方法：動態規劃
        時間複雜度: O(m * n)
        空間複雜度: O(m * n)
        """
        m, n = len(s), len(p)
        
        # dp[i][j]: s 的前 i 個字元與 p 的前 j 個字元是否匹配
        dp = [[False] * (n + 1) for _ in range(m + 1)]
        
        # 初始化：空字串匹配空模式
        dp[0][0] = True
        
        # 初始化：空字串匹配 x* 模式
        for j in range(2, n + 1):
            if p[j - 1] == '*':
                dp[0][j] = dp[0][j - 2]
        
        # 填充 DP 表格
        for i in range(1, m + 1):
            for j in range(1, n + 1):
                if p[j - 1] == '*':
                    # 情況 1: 匹配 0 次（忽略 x*）
                    dp[i][j] = dp[i][j - 2]
                    
                    # 情況 2: 匹配 1+ 次
                    if s[i - 1] == p[j - 2] or p[j - 2] == '.':
                        dp[i][j] = dp[i][j] or dp[i - 1][j]
                else:
                    # 普通字元或 '.'
                    if s[i - 1] == p[j - 1] or p[j - 1] == '.':
                        dp[i][j] = dp[i - 1][j - 1]
        
        return dp[m][n]
#include <string>
#include <vector>

class Solution {
public:
    /**
     * 方法：動態規劃
     * 時間複雜度: O(m * n)
     * 空間複雜度: O(m * n)
     */
    bool isMatch(string s, string p) {
        int m = s.length(), n = p.length();
        
        // dp[i][j]: s 的前 i 個字元與 p 的前 j 個字元是否匹配
        vector> dp(m + 1, vector(n + 1, false));
        
        // 初始化：空字串匹配空模式
        dp[0][0] = true;
        
        // 初始化：空字串匹配 x* 模式
        for (int j = 2; j <= n; j++) {
            if (p[j - 1] == '*') {
                dp[0][j] = dp[0][j - 2];
            }
        }
        
        // 填充 DP 表格
        for (int i = 1; i <= m; i++) {
            for (int j = 1; j <= n; j++) {
                if (p[j - 1] == '*') {
                    // 情況 1: 匹配 0 次（忽略 x*）
                    dp[i][j] = dp[i][j - 2];
                    
                    // 情況 2: 匹配 1+ 次
                    if (s[i - 1] == p[j - 2] || p[j - 2] == '.') {
                        dp[i][j] = dp[i][j] || dp[i - 1][j];
                    }
                } else {
                    // 普通字元或 '.'
                    if (s[i - 1] == p[j - 1] || p[j - 1] == '.') {
                        dp[i][j] = dp[i - 1][j - 1];
                    }
                }
            }
        }
        
        return dp[m][n];
    }
};
class Solution {
    /**
     * 方法：動態規劃
     * 時間複雜度: O(m * n)
     * 空間複雜度: O(m * n)
     */
    public boolean isMatch(String s, String p) {
        int m = s.length(), n = p.length();
        
        // dp[i][j]: s 的前 i 個字元與 p 的前 j 個字元是否匹配
        boolean[][] dp = new boolean[m + 1][n + 1];
        
        // 初始化：空字串匹配空模式
        dp[0][0] = true;
        
        // 初始化：空字串匹配 x* 模式
        for (int j = 2; j <= n; j++) {
            if (p.charAt(j - 1) == '*') {
                dp[0][j] = dp[0][j - 2];
            }
        }
        
        // 填充 DP 表格
        for (int i = 1; i <= m; i++) {
            for (int j = 1; j <= n; j++) {
                if (p.charAt(j - 1) == '*') {
                    // 情況 1: 匹配 0 次（忽略 x*）
                    dp[i][j] = dp[i][j - 2];
                    
                    // 情況 2: 匹配 1+ 次
                    if (s.charAt(i - 1) == p.charAt(j - 2) || p.charAt(j - 2) == '.') {
                        dp[i][j] = dp[i][j] || dp[i - 1][j];
                    }
                } else {
                    // 普通字元或 '.'
                    if (s.charAt(i - 1) == p.charAt(j - 1) || p.charAt(j - 1) == '.') {
                        dp[i][j] = dp[i - 1][j - 1];
                    }
                }
            }
        }
        
        return dp[m][n];
    }
}

複雜度分析

時間複雜度: O(m × n)

需要填充 m×n 的 DP 表格，其中 m 是字串長度，n 是模式長度。

空間複雜度: O(m × n)

需要 m×n 的 DP 表格存儲狀態。
可優化至 O(n)：只保留兩行（當前行和上一行）

重點提示

  • DP 狀態定義：dp[i][j] = s 前 i 字元匹配 p 前 j 字元
  • '*' 的兩種情況：
    • 匹配 0 次：dp[i][j-2]
    • 匹配 1+ 次：dp[i-1][j]（前提是字元匹配）
  • 初始化關鍵：
    • dp[0][0] = true（空匹配空）
    • dp[0][j] 處理 x* 模式（可以匹配空字串）
  • 常見錯誤：
    • 忘記初始化 dp[0][j]
    • * 的邏輯錯誤（要先考慮匹配 0 次）
    • 索引偏移錯誤（dp 是 1-indexed，字串是 0-indexed）
  • 面試技巧：
    • 先說明 DP 的狀態定義
    • 重點解釋 * 的處理邏輯
    • 用簡單例子演示 DP 表格
    • 提到空間優化可能性
