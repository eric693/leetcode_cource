2. Add Two Numbers
Medium

題目描述

給你兩個非空的鏈表，表示兩個非負的整數。它們每位數字都是按照逆序的方式存儲的，並且每個節點只能存儲一位數字。

請你將兩個數相加，並以相同形式返回一個表示和的鏈表。

你可以假設除了數字 0 之外，這兩個數都不會以 0 開頭。

範例 1:
輸入: l1 = [2,4,3], l2 = [5,6,4]
輸出: [7,0,8]
解釋: 342 + 465 = 807
範例 2:
輸入: l1 = [0], l2 = [0]
輸出: [0]
範例 3:
輸入: l1 = [9,9,9,9,9,9,9], l2 = [9,9,9,9]
輸出: [8,9,9,9,0,0,0,1]
限制條件:
每個鏈表中的節點數在範圍 [1, 100] 內
0 <= Node.val <= 9
題目數據保證列表表示的數字不含前導零

解題思路

方法：模擬加法過程（推薦）⭐

就像我們手算加法一樣，從最低位開始，逐位相加，處理進位。

核心思想：

使用一個虛擬頭節點（dummy node），遍歷兩個鏈表，將對應位置的數字相加，並處理進位。

步驟：

  1. 創建虛擬頭節點 dummy 和當前節點指標 current
  2. 初始化進位 carry = 0
  3. 當 l1、l2 或 carry 不為 0 時：
    • 取得當前位的值：val1 (l1 的值或 0)、val2 (l2 的值或 0)
    • 計算總和：total = val1 + val2 + carry
    • 更新進位：carry = total / 10
    • 創建新節點：total % 10
    • 移動指標到下一位
  4. 返回 dummy.next（跳過虛擬頭節點）

視覺化範例：l1 = [2,4,3], l2 = [5,6,4]

    2 → 4 → 3    (表示 342)
+   5 → 6 → 4    (表示 465)
─────────────
    7 → 0 → 8    (表示 807)

步驟 1: 2 + 5 = 7, carry = 0
       創建節點 7

步驟 2: 4 + 6 = 10, carry = 1
       創建節點 0

步驟 3: 3 + 4 + 1(carry) = 8, carry = 0
       創建節點 8

結果: 7 → 0 → 8

進位示例：l1 = [9,9,9,9,9,9,9], l2 = [9,9,9,9]

    9 → 9 → 9 → 9 → 9 → 9 → 9
+   9 → 9 → 9 → 9
─────────────────────────────
    8 → 9 → 9 → 9 → 0 → 0 → 0 → 1

注意: 最後還需要處理進位 1
關鍵技巧
  • 虛擬頭節點：簡化邊界處理，避免特殊判斷第一個節點
  • 進位處理：carry = total / 10，當前位 = total % 10
  • 長度不一：當某個鏈表結束時，將其值視為 0
  • 最後進位：循環條件包含 carry，確保最後的進位被處理

程式碼實現

class Solution:
    def addTwoNumbers(self, l1: Optional[ListNode], l2: Optional[ListNode]) -> Optional[ListNode]:
        """
        方法：模擬加法過程
        時間複雜度: O(max(m, n))
        空間複雜度: O(max(m, n))
        """
        # 創建虛擬頭節點
        dummy = ListNode(0)
        current = dummy
        carry = 0
        
        # 當任一鏈表未結束或還有進位時
        while l1 or l2 or carry:
            # 取得當前位的值
            val1 = l1.val if l1 else 0
            val2 = l2.val if l2 else 0
            
            # 計算總和
            total = val1 + val2 + carry
            
            # 更新進位和當前位
            carry = total // 10
            current.next = ListNode(total % 10)
            
            # 移動指標
            current = current.next
            if l1:
                l1 = l1.next
            if l2:
                l2 = l2.next
        
        return dummy.next

 * Definition for singly-linked list.
 * struct ListNode {
 *     int val;
 *     ListNode *next;
 *     ListNode() : val(0), next(nullptr) {}
 *     ListNode(int x) : val(x), next(nullptr) {}
 *     ListNode(int x, ListNode *next) : val(x), next(next) {}
 * };
 */

class Solution {
public:
    /**
     * 方法：模擬加法過程
     * 時間複雜度: O(max(m, n))
     * 空間複雜度: O(max(m, n))
     */
    ListNode* addTwoNumbers(ListNode* l1, ListNode* l2) {
        // 創建虛擬頭節點
        ListNode* dummy = new ListNode(0);
        ListNode* current = dummy;
        int carry = 0;
        
        // 當任一鏈表未結束或還有進位時
        while (l1 != nullptr || l2 != nullptr || carry != 0) {
            // 取得當前位的值
            int val1 = (l1 != nullptr) ? l1->val : 0;
            int val2 = (l2 != nullptr) ? l2->val : 0;
            
            // 計算總和
            int total = val1 + val2 + carry;
            
            // 更新進位和當前位
            carry = total / 10;
            current->next = new ListNode(total % 10);
            
            // 移動指標
            current = current->next;
            if (l1 != nullptr) l1 = l1->next;
            if (l2 != nullptr) l2 = l2->next;
        }
        
        return dummy->next;
    }
};

 * Definition for singly-linked list.
 * public class ListNode {
 *     int val;
 *     ListNode next;
 *     ListNode() {}
 *     ListNode(int val) { this.val = val; }
 *     ListNode(int val, ListNode next) { this.val = val; this.next = next; }
 * }
 */

class Solution {
    /**
     * 方法：模擬加法過程
     * 時間複雜度: O(max(m, n))
     * 空間複雜度: O(max(m, n))
     */
    public ListNode addTwoNumbers(ListNode l1, ListNode l2) {
        // 創建虛擬頭節點
        ListNode dummy = new ListNode(0);
        ListNode current = dummy;
        int carry = 0;
        
        // 當任一鏈表未結束或還有進位時
        while (l1 != null || l2 != null || carry != 0) {
            // 取得當前位的值
            int val1 = (l1 != null) ? l1.val : 0;
            int val2 = (l2 != null) ? l2.val : 0;

            int total = val1 + val2 + carry;
            
            // 更新進位和當前位
            carry = total / 10;
            current.next = new ListNode(total % 10);

            current = current.next;
            if (l1 != null) l1 = l1.next;
            if (l2 != null) l2 = l2.next;

    }
}

複雜度分析

時間複雜度: O(max(m, n))

需要遍歷兩個鏈表中較長的那個，其中 m 和 n 分別是兩個鏈表的長度。

空間複雜度: O(max(m, n))

新建的鏈表長度最多為 max(m, n) + 1（可能有最後的進位）。

重點提示

  • 虛擬頭節點技巧：避免特殊處理第一個節點，代碼更簡潔
  • 進位處理：
    • 當前位的值 = (val1 + val2 + carry) % 10
    • 新的進位 = (val1 + val2 + carry) / 10
  • 長度不同處理：當某個鏈表結束時，將其值視為 0
  • 最後的進位：循環條件必須包含 carry，例如 999 + 1 = 1000
  • 邊界情況：
    • 兩個鏈表長度相同
    • 兩個鏈表長度不同
    • 最後有進位（如 99 + 1 = 100）
    • 其中一個鏈表為空（題目說明不會發生）
  • 常見錯誤：
    • 忘記處理最後的進位
    • 循環條件不正確（應該是 or 而非 and）
    • 沒有使用虛擬頭節點，導致代碼複雜
  • 面試技巧：
    • 先說明思路：模擬手算加法過程
    • 強調虛擬頭節點的好處
    • 說明如何處理長度不同和進位
    • 可以討論如果數字是正序存儲該如何處理
