11. Container With Most Water
Medium

題目描述

給定 n 個非負整數 a1, a2, ..., an，每個數代表座標中的一個點 (i, ai)。在座標內畫 n 條垂直線，垂直線 i 的兩個端點分別為 (i, ai) 和 (i, 0)。

找出其中的兩條線，使得它們與 x 軸共同構成的容器可以容納最多的水。

注意：你不能傾斜容器。

範例 1:
輸入: height = [1,8,6,2,5,4,8,3,7]
輸出: 49
解釋: 圖中垂直線代表輸入陣列 [1,8,6,2,5,4,8,3,7]
     在此情況下，容器能夠容納水（藍色部分）的最大值為 49
範例 2:
輸入: height = [1,1]
輸出: 1

解題思路

方法一：暴力解法

嘗試所有可能的配對，計算每對線條能形成的面積，找出最大值。

時間複雜度：O(n²) - 不推薦

方法二：雙指標（推薦）

使用兩個指標從陣列兩端開始，逐步向中間移動，每次移動較短的線。

核心思想：

容器的面積由「較短的線」和「兩線之間的距離」決定。要最大化面積，我們從最大寬度開始，然後逐步嘗試更高的線。

為什麼移動較短的線？

  • 面積 = min(height[left], height[right]) × (right - left)
  • 移動較短的線，可能會遇到更高的線，從而增加面積
  • 移動較高的線，由於寬度減小且高度受限於較短的線，面積一定不會增加

步驟：

  1. 初始化 left = 0，right = n - 1，max_area = 0
  2. 當 left < right 時：
    • 計算當前面積：width = right - left
    • area = min(height[left], height[right]) × width
    • 更新 max_area
    • 如果 height[left] < height[right]，left++
    • 否則，right--
  3. 返回 max_area

程式碼實現

class Solution:
    def maxArea(self, height: List[int]) -> int:
        """
        雙指標解法
        時間複雜度: O(n)
        空間複雜度: O(1)
        """
        left, right = 0, len(height) - 1
        max_area = 0
        
        while left < right:
            # 計算當前面積
            width = right - left
            current_height = min(height[left], height[right])
            area = current_height * width
            
            # 更新最大面積
            max_area = max(max_area, area)
            
            # 移動較短的線
            if height[left] < height[right]:
                left += 1
            else:
                right -= 1
        
        return max_area
#include <vector>
#include <algorithm>

class Solution {
public:
    /**
     * 雙指標解法
     * 時間複雜度: O(n)
     * 空間複雜度: O(1)
     */
    int maxArea(std::vector<int>& height) {
        int left = 0, right = height.size() - 1;
        int maxArea = 0;
        
        while (left < right) {
            // 計算當前面積
            int width = right - left;
            int currentHeight = std::min(height[left], height[right]);
            int area = currentHeight * width;
            
            // 更新最大面積
            maxArea = std::max(maxArea, area);
            
            // 移動較短的線
            if (height[left] < height[right]) {
                left++;
            } else {
                right--;
            }
        }
        
        return maxArea;
    }
};
class Solution {
    /**
     * 雙指標解法
     * 時間複雜度: O(n)
     * 空間複雜度: O(1)
     */
    public int maxArea(int[] height) {
        int left = 0, right = height.length - 1;
        int maxArea = 0;
        
        while (left < right) {
            // 計算當前面積
            int width = right - left;
            int currentHeight = Math.min(height[left], height[right]);
            int area = currentHeight * width;
            
            // 更新最大面積
            maxArea = Math.max(maxArea, area);
            
            // 移動較短的線
            if (height[left] < height[right]) {
                left++;
            } else {
                right--;
            }
        }
        
        return maxArea;
    }
}

複雜度分析

時間複雜度: O(n)

兩個指標各遍歷陣列一次，總共 O(n) 時間。

空間複雜度: O(1)

只使用了常數額外空間。

重點提示

  • 貪心策略：從最大寬度開始，逐步尋找可能更高的線
  • 移動策略：總是移動較短的線，因為只有這樣才可能增加面積
  • 證明正確性：移動較高的線一定不會得到更大的面積（寬度減小，高度受限）
  • 面積公式：area = min(height[left], height[right]) × (right - left)
  • 常見錯誤：不要嘗試同時移動兩個指標
